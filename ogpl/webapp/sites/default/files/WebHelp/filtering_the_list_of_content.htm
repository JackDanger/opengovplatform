Filtering the Content List

To view a list of specific content, you can specify various filter criteria, such as the type of content or the current status of the content in the workflow.

To filter the content list:

  1. In the navigation menu, click the Home link.

  2. If you want to view information for a specific type of content, in the Content type list, click the appropriate content type.
    If you want to view information for a specific workflow step, in the Workflow step list, click the appropriate workflow step.
    For example, if you want to view the list of FAQ content that is currently published on the OGPL Web site, in the Content type list, click FAQ, and then in the Workflow step list, click Published.
    Tip: To select multiple filters, hold down the CTRL key and click each content type or workflow step.

  3. Click Apply.
    The Content List section displays the list of content that matches the filter criteria that you specify.
