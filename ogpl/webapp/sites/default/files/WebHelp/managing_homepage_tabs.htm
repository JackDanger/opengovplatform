Managing Homepage Tabs

Homepage Tabs allow you to display tabbed content on the home page of the OGPL Web site. Homepage Tabs appear on the home page of the OGPL Web site when the Super Administrator changes the Rotating Panel from full to half banner .

 

The Front End Home Page Rotating Tabs is a Drupal Quicktab block which contains 3 Homepage Tabs. Homepage Tabs use Views to fetch information from the Drupal content source and then display the dataset lists in slide show fashion on the home page of the OGPL Web site:

Table: Description of Hompage Tabs and their content sources

Homepage Tab

Description and Content Source

Popular Catalogs

Displays most-viewed Apps catalogs, dataset catalogs, and document catalogs.

This Homepage Tab uses the Most_Popular_Catalog view as the content source. The Most_Popular_Catalog view in turn uses the following views to fetch the information:

  • Popular_Apps_Catalog

  • Popular_Dataset_Catalog

  • Popular_Document_Catalog

Recent Catalogs

Displays recently added Apps catalogs, dataset catalogs, and document catalogs.

This Homepage Tab uses the Most_Recent_Catalogs view as the content source. The Most_Recent_Catalog view in turn uses the following views to fetch the information:

  • Recent_Apps_Catalog

  • Recent_Dataset_Catalog

  • Recent_Document_Catalog

Major Events

Displays major events happening around the world through the RSS feeds.

This Homepage Tab uses a block as a content source. For example, RSS feed aggregator. For more information, see Managing RSS Feeds
