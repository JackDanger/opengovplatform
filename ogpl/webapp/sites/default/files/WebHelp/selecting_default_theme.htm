Selecting Default Theme

You can set the default theme for the users of the OGPL Control Panel application and the OGPL Web site, by performing the following steps:

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Themes.
    The Themes page appears with a list of the all the themes currently available in the OGPL Control Panel.

  2. In the row of the theme that you want to set as default, select the check box in the Default column.

  3. Click Save configuration.
    The theme that you selected is saved as the default theme for the OGPL Control Panel application and the OGPL Web site.
