Resetting Site Information

When the updated site configuration settings results in an error or show undesirable results, you can reset all the site configuration settings to their default values.

To reset the site configuration to their default values:

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Site Information.
    The Site Information page appears.

  2. Scroll down until you reach the end of the page, and then click Reset to defaults.
    All the site configuration settings are reset to their default values.
