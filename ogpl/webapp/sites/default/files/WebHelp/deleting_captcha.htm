Deleting Captcha

You can remove a CAPTCHA from an OGPL Web page by performing the following steps:

  1. In the navigation menu, point to Configurable Items, and then click Manage Captcha.
    The General Settings tab displays global settings for the CAPTCHA that you selected.

  2. If you only want to hide the CAPTCHA on a Web page, without deleting it, in the Form protection section, in the row of the Web page for which you want to remove the CAPTCHA, click the none.
    The CAPTCHA is removed from the Web page.
    -or-
    If you want to delete the CAPTCHA from a Web page, in the Form protection section, in the row of the Web page for which you want to delete the CAPTCHA, click the delete link. A confirmation message appears.
    Click Delete.

The CAPTCHA is deleted from the Web page.
