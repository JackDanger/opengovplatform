Filtering the Feedback List

To view specific feedback from users of the OGPL Web site, you can specify various filter criteria, such as the feedback category, start date, end date, or source.

Note: In the My List tab, you can also filter the feedback list by the status of feedback.

 

To filter the list of feedback:

  1. In the navigation menu, click the VRM link.
    The VRM page appears.

  2. Click the tab in which you want to sort the feedback list.

  3. Click the Filters link.
    The Filters section expands.

  4. Select one or more filter criteria. The following table lists each field and its description.
    Table: Description of fields in the Filters section

Title

Description

Category

Click the feedback category for which you want to view the list of feedback received.

For example, if you want to view the list of feedback received related to tools, click Tools.

Note: To select multiple categories, hold down the CTRL key and click each category.

From date

Click the start date of the duration for which you want to view the list of feedback received.

By default, this field displays the date 30 days prior to the current date.

For example, if you want to view the list of feedback received between January 1, 2012 and January 30, 2012, click January 1, 2012.

To date

Click the end date of the duration for which you want to view the list of feedback received.

By default, this field displays the current date as the To date.

For example, if you want to view the list of feedback received between January 1, 2012 and January 30, 2012, click January 30, 2012.

Source

Click the name of the source for which you want to view the list of feedback received.

For example, if you want to view the list of feedback received from the Contact Us page, click Contact Us.

Status

Click the feedback status for which you want to view the list of feedback received.

For example, if you want to view the list of feedback assigned to an owner, click Assigned.

Note: This filter criteria is only available in the My List tab.

 

  1. Click Apply.
    The feedback matching the filter criteria appears on the feedback list.
