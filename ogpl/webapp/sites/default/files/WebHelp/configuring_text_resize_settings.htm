Configuring Text Resize Settings

Accessibility options help visitors who have visual disabilities to view text on the OGPL Web site. Using these options, a visitor can increase or decrease the size of the text on the OGPL Web page. You can also specify whether the text size updates must be applied across the entire OGPL Web site or within a particular section, such as a paragraph or a table.

To configure text resize options:

  1. In the navigation menu, point to the Configurable Items, Manage Header Content, and then click Manage Accessibility (Text Resize).
    The Text Size Configuration page appears.

  2. Configure the appropriate options. The following tables list each field and its description.

Global settings

Table: Description of fields in the Global settings section of Text Size Configuration page

Field

Description

Block title

Click the title that must appear as the heading for the text resize section on the OGPL Web site. By default, the block title is set to Text Size.

Block type

Click the type of block in which you want to display the text resize options. By default, the Block type is set to Text links, which displays three text hyperlinks (Decrease, Normal and Increase) on the OGPL Web site.

If you want to display the text resize options as images, click Image links. This setting provides a good user experience for Web Accessibility.

If you want to display a list from which a visitor can choose a text size, click Select menu.

Type of links

Click the text increase option that you want to provide to the visitors of the OGPL Web site. The text increase option determines how the visitors can increase or decrease the text size on the OGPL Web site.

By default, the Type of links is set to Variable. This allows the visitors to increase or decrease the text size multiple times, until it reaches the text size limit that you specify in the Advanced settings.

If you want the visitors to have only standard text resize options, which is Decrease, Normal, and Increase, clicks Fixed. The visitors can only decrease or increase the text size once. The percentage of text size update depends on the value that you specify in the Advanced settings.

 

Advanced settings

Table: Description of fields in the Advanced settings section of Text Size Configuration page

Field

Description

XHTML element

Click the XHTML code element for which you want to apply the text resize updates. By default, the XHTML element is set to div and the text resize update is applied to all text content within the OGPL Web site. Considering the complete html content is built in the main div tags, we have selected “div” as the default option.

XHTML element ID/class

Enter an ID, a class name, or a comma-separated list of IDs or classes for the XHTML element to which you want to apply the text resize updates. By default, this field is blank and the text resize update is applied to all text content within the OGPL Web site.

 

For example, if you want to apply text resize update to the content within the ID named myid, in the XHTML element ID/class box, enter #myid.

 

Note: If you want to enter a list of IDs in this field, you must clear the Overwrite text size in templates check box.

Increases/decreases step (Required)

Enter a percentage value of the current text size, between 1% and 99%, by which you want to increase or decrease the text size at each step.
By default, the Increases/decreases step is set to 5%, which increases the current text size by 5% of the current text size each time that a visitor zooms in or zooms out.

Minimum text size (Required)

Enter a percentage value of the current text size, between 1% and 998%, below which a visitor cannot decrease the text size on the OGPL Web site.

By default, the Minimum text size is set to 50% of the current text size on the OGPL Web site.

Normal text Size (Required)

Enter the default percentage value of the current text size, between 2% and 999%, at which you want to display the OGPL Web site text.
By default, the Normal text Size is set to 75% of the current text size on the OGPL Web site.

Display normal text size as 100%

If you want to display the text on the OGPL Web site at 100% of the current text size, select this check box.

Reset to normal size after maximum or minimum reached

If a visitor tries to increase or decrease the text size beyond the limit and you want to update the size of the text to the Normal Text Size value, select this check box.


For example, the Minimum text size value is 30% of the current text size, the Maximum text size value is 90%, and the Normal text size value is 75%. If a visitor tries to resize the text to less than 30% or more than 90%, the text is updated to 75% of the current text size on the OGPL Web site.

Maximum text size (Required)

Enter a percentage value of the current text size, between 3% and 1000%, above which a visitor will not be able to increase the text size on the OGPL Web site.

By default, the Maximum text size is set to 150% of the current text size on the OGPL Web site.

Support JavaScript

If you want to update the text size on the OGPL Web site without reloading the complete Web page, select this check box.
By default, this check box is selected.

 

If you want to update the text size on the OGPL Web site by reloading the complete Web page, clear this check box.

Overwrite text size in templates

If you want to overwrite the font-size value of the current theme with the values that you specify in the Text Resize Configuration page, select this check box. If there is any conflict in the font size settings, the values that you specify in the Text Resize Configuration page override the values in the current theme.

If you do not want to overwrite the font-size value in the current theme with the values that you specify in the Text Resize Configuration page, clear this check box. However, if you selected “body” option in the XHTML element field, then you must delete the section "body {font-size: ...;}" from the current theme of the OGPL Web site.  

Display message after changing text size

If you want to display the updated text size on the OGPL Web site when a visitor updates the text size, select this check box.

Cookie Expires (Required)

Enter the number of days, between 1 and 3650, after which the OGPL Web site cookies should expire. By default, the cookies expire after 365 days.

Cookie Domain  (Required)

Enter the OGPL domain URL for which you want to store the OGPL cookie on the computer of the visitor who is accessing the URL.  By default, this value is set to the OGPL Web site URL. If you change this setting, visitors must access the new URL in order to store the cookie on their computer.

 

Theme settings

Table: Description of fields in the Theme settings section of Text Size Configuration page

Field

Description

Subtitle

If you want to display the title that you selected in the Block title section in Global settings, select this check box. This setting is recommended for Image links and Select menu block types.

If you do not want to display any title for the text resize section on the OGPL Web site, clear this check box.

Increase Text Size

If you want to display the increase text size option on the OGPL Web site, select this check box.

Decrease Text Size

If you want to display the decrease text size option on the OGPL Web site, select this check box.

Normal Text Size

If you want to display the option to change the text size to the value you that you specify in the Normal Text Size field, select this check box.

Link text

If you want to display the text resize labels on the OGPL Web site, select this check box. You cannot use this option for the Image links block type.

Display current text size text and value

If you want to display the current text size with a label on the OGPL Web site, click Text and value (for example: "Current Size: 100%")


If you want to display only the current text size (without a label) on the OGPL Web site, click Only the value (for example: "100%").


If you want to display neither the current text size nor a label on the OGPL Web site, click Hide text and value.

If you want to remove the current text size label and value from the XHTML source code, click Remove text and value in the source code.

Display inline

Use this section to specify whether the text resize options and their labels should appear in the same row on the OGPL Web site.

Subtitle, list and select menu

If you want to display the text resize options and their labels (that you selected in the Subtitle section) in the same row on the OGPL Web site, select this check box.  

Current text size

If you want to display the current text size in the same row as the text resize options on the OGPL Web site, select this check box.

 

  1. Click Save configuration.
    The text resize options appear on the OGPL Web site.
