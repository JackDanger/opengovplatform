Viewing Captcha Examples

You can browse through examples of all available CAPTCHA types by performing the following steps:

  1. In the navigation menu, point to Configurable Items, and then click Manage Captcha.
    The     CAPTCHA page appears.

  2. Click the Examples tab.  
    The     Examples tab displays one example for the Math CAPTCHA, Image CAPTCHA and the reCAPTCHA.

  3. To view ten more examples of a particular CAPTCHA type, click 10 more examples of this challenge.
