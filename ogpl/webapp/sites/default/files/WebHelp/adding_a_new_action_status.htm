Adding a New Action Status

If you want to use an action status that is not available in the Action Status List section, you can add a new action status by performing the following steps:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Manage Actions tab.
    The Action Status List page appears.

  3. Click the Add tab.
    The Add Action Status page appears.

  4. In the Action Status Name (Required) box, enter the appropriate name for this action status. For example, if you want to create an action status for the feedback that needs immediate attention, enter Actionable-Urgent.

  5. Click Add.
    The new action status appears in the Action Status List section. You can now use this action status for a feedback.
