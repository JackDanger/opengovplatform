Marking a Feedback as Irrelevant

When you mark a feedback as irrelevant, the status of that feedback is updated to Irrelevant, but the feedback remains in the database.

Notes:

  • You can only mark a feedback that is in the Open state as irrelevant.

  • If a feedback is marked irrelevant, you cannot reuse the feedback.

To mark a feedback as irrelevant:

  1. In the navigation menu, click the Home link.

  2. Click the Opened tab.
    A page that displays the list of feedback that is in the Opened status appears.

  3. Optional: You can filter the feedback list. For more information, see Filtering the List of Feedback.

  4. Click the Edit this record icon for the feedback that you want to mark as irrelevant.
    The Feedback Details Edit page appears.

  5. Click the appropriate category for the feedback.

  6. Scroll down until you reach the end of the page, and then click Move to “Irrelevant”.

The status of the feedback is updated to Irrelevant.
