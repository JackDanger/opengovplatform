Configuring Date and Time Format Settings

You can configure the format of the date and time that must be displayed on the OGPL Web site by performing the following steps:

  1. In the Drupal menu, point to Site configuration, and click Date and time.
    The Date And Time page appears.

  2. Click the Formats tab.

  1. In the Date formats section, click the appropriate information. The following table lists each field and its description:

Table: Description of options in the Dates format section

Title

Description

Long date format

Click the appropriate long date format that you want to display on the OGPL Web site.

Medium date format

Click the appropriate medium date format that you want to display on the OGPL Web site.

Short date format

Click the appropriate short date format that you want to display on the OGPL Web site.

 

  1. If you want to display another date format on the OGPL Web site, in the Add format type section, you can add the appropriate date format.

  2. Click Save configuration.
    The date and time format is saved.

Tip: You can also create a date and time format from the Custom Formats and Add Format tabs.
