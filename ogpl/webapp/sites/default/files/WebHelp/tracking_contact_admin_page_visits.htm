Tracking Contact Admin Page Visits

You can view and track the visits to the Contact Admin page by performing the following steps:

  1. In the navigation menu, click Contact Admin.
    The Contact Admin page appears.

  2. Click the Track tab.
    The Track tab displays the information about the visits to the Contact Admin page. The following table lists each field and its description.

Table: Description of fields on the Track page

Column

Description

Time

Displays the date and time when an OGPL Administrator accessed the Contact Admin page.

Referrer

Displays URL of the Web page from where the OGPL Administrator accessed the Contact Admin page.

User

Displays the username of the OGPL Administrator that accessed the Contact Admin page.

Operations

Click the details link to view information about the changes made on the Contact Admin page. You can view the following information:

  • URL of the Contact Admin page.

  • Title of the Contact Admin page.

  • URL of the page from where OGPL Administrator accessed the Contact Admin page.

  • Date and time when the OGPL Administrator accessed the Contact Admin page.

  • Username of the OGPL Administrator that accessed the Contact Admin page.

  • IP address of the computer that was used to access the Contact Admin page.
