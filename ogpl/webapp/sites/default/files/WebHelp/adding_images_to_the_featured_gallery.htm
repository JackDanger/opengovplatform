Adding Images to a Featured Gallery

The Featured Gallery displays a slide show of dataset images that you want to highlight on the OGPL Web site. You or a Content Creator can upload an image of a dataset to the Featured Gallery, and link the image to the appropriate dataset. The Moderator reviews the content to verify whether the content is appropriate for the OGPL Web site. When a visitor clicks an image on the Featured Gallery, the appropriate dataset page appears and displays information about the dataset.

Notes:

  • You must have an image of the dataset.

  • The recommended image dimension is 500 x 292 pixels.

 

The following image illustrates how Featured Gallery appears on the OGPL Web site.

To upload an image to the Featured Gallery:

  1. In the navigation menu, point to Configurable Items, and then click Create Content.
    The Create Content page appears.

  2. Click the Featured Gallery link.
    The Create Featured Gallery page appears.

  3. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Create Featured Gallery page

Title

Description

Title

(Required)

Enter an appropriate title for this featured gallery image.

Menu settings

You do not need to provide information for fields within this section.

Revision information -

Log message

Enter the reason for creating this dataset.

URL path settings

By default, the Automatic alias check box is selected. The application automatically assigns the URL for this image.

If you want to assign a specific URL for this image, perform the following steps:

  1. Click to clear the Automatic alias check box.

  2. In the URL path settings box, type the URL that you want to assign for this image.

Image

Upload an image of the dataset.

To upload an image:

  1. Click Browse.
    The File Upload page appears.

  2. Click the image that you want to upload to the dataset.

  3. Click Open.
    The Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded to the database and a thumbnail of the image appears on the page.

  5. In the Alternate Text field, type a description of the image.
    This description is read by screen readers and search engines. If the image fails to load, this description appears on the OGPL Web site.

  6. In the Title field, type an alternate title for the image.
    This text appears as a screentip for the image.

  7. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

High Value Datasets

Displays the list of datasets that are currently on the OGPL Web site.

Because you are adding an image for a dataset that is not yet created, click N/A.

The Content Creator will create a dataset and link it to the image.
