Viewing the List of Themes

You can view the list of themes that are available in the OGPL Control Panel by performing the following steps:

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Themes.
    The     Themes page appears with the list of the themes that are available in the OGPL Control Panel.

  2. Browse through the list of themes. The following table lists each field and its description.  

Table: Description of fields in the Themes page

Field

Description

Screenshot

Displays a visual representation for this theme.

Name

Displays the theme name and the description for this theme.

Version

Displays the current version of this theme.

Enabled

Indicates whether this theme is available to use in the OGPL Control Panel application. There can be multiple themes enabled at a time.

Default

Indicates whether this theme is the default theme for the OGPL Control Panel application and the OGPL Web site.

Operations

Displays the configure link that allows you to customize this theme. For more information, see Configuring Themes
