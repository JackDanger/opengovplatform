Deleting a Feedback Category

You can delete an inappropriate feedback category that is not associated with any feedback. When you delete a feedback category, all the details associated with the feedback category, such as the predefined text, are deleted from the database.

Note: You cannot delete the Website Related, App and Raw Data feedback categories.

To delete a feedback category:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Manage Categories tab.
    The Category List page appears.

  3. In the Operations menu, click the edit link for the feedback category that you want to delete.
    The Edit Category page appears.

  4. Click Delete.
    The Edit Category page appears and displays a confirmation message.

  5. Click Delete.
    The feedback category is removed from the Category List.
