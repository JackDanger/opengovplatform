Adding Notes to a Feedback

You can add notes to a feedback while assigning a feedback to an owner or while adding general information about the feedback, such as log information. The feedback owner can read the notes and take the appropriate action required.

To add notes to a feedback:

  1. In the navigation menu, click the Home link.

  2. Click the status tab that corresponds to the current status of the feedback for which you want to add a note. For example, if you want to add a note to a feedback that is in Open status, click the Opened tab.

  3. Optional: You can filter the feedback list. For more information, see Filtering the List of Feedback.

  4. In the row of the feedback for which you want to add a note, click the Edit this record icon .
    The Feedback Details Edit page appears.

  5. Click the Notes tab.
    The Notes page appears.

  6. In the Add note box, enter the notes for this feedback.

  7. Click Submit.

The note is added to the feedback.
