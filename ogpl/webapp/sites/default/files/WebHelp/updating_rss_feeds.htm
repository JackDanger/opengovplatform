Updating RSS Feeds

You can update the RSS feed of a Web site added to the OGPL Web site, by updating the RSS feed name, the URL for the feed, or the interval at which the RSS feed is updated.

To update a RSS feed:

  1. In the navigation menu, point to Configurable Items, and then click Manage RSS Aggregator.
    The Feed Aggregator page appears.

  2. In the row of the RSS feed that you want to update, under the Operations column, click the edit link.
    The Edit Feed page opens.  

  3. In the Title field, update the name of the source from which you want to receive the RSS feed.
    Note: The name that you enter in the Title field appears on the OGPL Web site.

  4. In the URL field, update the complete URL for the RSS feed.

  5. In the Update Interval list, click the appropriate time interval between each RSS feed update.
    Note: The interval at which RSS feed is updated also depends on the interval that you specified in Automatically run cron section in the Site Information Web page.  For more information, see Configuring the Site Information.
