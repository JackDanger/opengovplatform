Viewing the Reply for a Feedback

Each reply is logged in the application so that the reply is available for further analysis or review.

To view the reply for a feedback:

  1. In the navigation menu, click the Home link.

  2. Click the Replied tab.
    A page that displays the list of feedback for which replies have been sent appears.

  3. Optional: You can filter the feedback list. For more information, see Filtering the list of Feedback.

  4. In the row of the feedback for which you want to view the reply, click the Preview this record icon .
    The Feedback Details page appears.

  5. Click the Reply / Review tab.

A page that displays the reply for the feedback, the username of the person who replied to the feedback, and the date when the reply was sent appears.
