Managing the RSS Feed Aggregator

RSS stands for Really Simple Syndication. RSS allows you to track when a Web site or blog is updated.

RSS can help a Web site by:

  • Delivering current information from the Internet in your areas of interest.

  • Providing content for Web sites.

  • Bringing additional traffic to a Web site.

The following image illustrates how RSS Feed Aggregator appears on the OGPL Web site.

You can subscribe to an RSS feed of a Web site and add it to the OGPL Web site by performing the following steps:

  1. In the navigation menu, point to Configurable Items, and then click Manage RSS Aggregator.
    The Feed Aggregator page appears.

  2. Under the Operations column, click the edit link.
    The Edit Feed page opens.

  3. In the Title field, update the name of the Web site from which you want to get the RSS feed.

  4. In the URL field, update the URL of the RSS feed.

  5. In the Update Interval list, click the appropriate time interval between each RSS feed update.

  6. Click Save.

The RSS feed aggregator is updated.
