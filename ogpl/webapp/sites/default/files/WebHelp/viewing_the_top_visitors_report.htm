Viewing the Top Visitors Report

The Top Visitors report lists the IP address of the visitors and the number of times that each visitor accessed a page on the OGPL Web site. This report also lists the total  time that the OGPL Web site took to load these pages.

To view the Top Visitors report, in the navigation menu, point to Reports, CMS Reports, Access Statistics, and then click Top Visitors.

By default, the IP address of the visitors that visited the OGPL Web site the most number of times appears at the top of the Top Visitors report. You can sort the records by clicking the appropriate column name.
