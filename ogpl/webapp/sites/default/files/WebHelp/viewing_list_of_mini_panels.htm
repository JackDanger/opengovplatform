Viewing the List of Mini Panels

You can view the list of Mini Panels that are currently configured in the OGPL Web site by performing the following steps:

  1. In the Drupal menu, point to Site building, Mini panels, and then click List.
    The Mini Panels page appears.
    The List tab displays all the Mini Panels that are configured in the OGPL Web site. The following table lists each field and its description.

Table: Description of fields in the List tab on Mini Panels page

Field

Description

Title

Displays the name of this Mini Panel as it appears on the OGPL Web site.

Name

Displays the name of this Mini Panel that you specified when creating the Mini Panel. This name does not appear on the OGPL Web site and it is only used in the OGPL Control Panel application.

Category

Displays the category to which the Mini Panel belongs.

Layout

Displays the type of layout for this Mini Panel.

Storage

Displays the state for this Mini Panel, based on the configuration in the Features section and in the OGPL database. The possible states are Default, Normal and Overridden.

If you created this Mini Panel using the defaults settings in Drupal, the Storage is set to Default.

If you created this Mini Panel using the Features module of Drupal, the Storage is set to Normal.

If you created this Mini Panel by modifying some of options in the Features module of Drupal, the Storage is set to Overridden.

Tip: If you revert the updates that you made for this Mini Panel in the Features module, the Storage is set to Normal.

Operations

Displays various tasks that you can perform with this Mini Panel, such as editing the Mini Panel, disabling the Mini Panel, enabling the Mini Panel, cloning the Mini Panel, or exporting the Mini Panel.
