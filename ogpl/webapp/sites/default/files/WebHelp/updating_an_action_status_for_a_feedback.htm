Updating an Action Status for a Feedback

If you want to update the name of an action status, you can edit the action status. When you update the name of an action status that was already assigned to some feedback, the action status name is automatically updated for all those feedback. For example, there are 5 feedback that are in the Actionable action status. When you update the action status named Actionable to Actionable-Urgent, the action status for those 5 feedback are automatically updated to Actionable-Urgent.

To update an action status:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Manage Actions tab.
    The Action Status List page appears.

  3. In the Operations column, click the edit link for the action status that you want to update.
    The Edit Action Status page appears.

  4. In the Action Status Name (Required) box, enter the new name for this action status.

  5. Click Save.
    The name of the action status is updated in the Action Status List section. If this action status was already assigned to some feedback, the action status name is automatically updated for all those feedback.
