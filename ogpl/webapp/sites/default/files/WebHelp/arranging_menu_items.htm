Arranging Menu Items

You can modify the sequence in which the menus and their menu-items are listed on the OGPL Web site. You can also change the hierarchy of menus and the sub-menus.

To rearrange the menu items:

  1. In the navigation menu, point to Configurable Items and then click Manage Menu.
    The Menus page appears.

  2. In the List menus tab, click the menu for which you want to rearrange the menu items.
    The List items tab displays the list of menu items available in menu that you selected.

  3. To rearrange a menu item, grab its drag-and-drop handle   by holding down the left mouse button while hovering over the handle icon.

  4. Drag the mouse pointer to the new location for the menu item in the list and release the left mouse button.
    The menu item moves to the new location.

  5. Optional: You can change the hierarchy of the menu items by moving a menu-item under another menu-item.
    Example: To make Accessibility menu-item a sub-menu of the Help menu item, move the Accessibility menu-item under the Help menu item.

  6. Click Save configuration.
    The OGPL Web site shows the updated menu item sequence and hierarchy.
