Updating the Cron Run Time Interval

By default, the cron run time interval is set to one hour. You can update the time interval between two cron runs. When you update the cron run time interval, the interval is applicable considering the last time when the cron was successfully executed.

For example, if the last time that the cron was successfully executed was at 1 PM, and you update the cron run time interval from one hour to three hours, the next cron will be executed at 4 PM.

To update the time interval:

  1. In the Drupal menu, point to Site Configuration, and then click Site Information.
    The Site Information page appears.

  2. Scroll down until you reach the end of the page.

  3. In the Automatically Run Cron list, click the appropriate time interval between two cron runs.

  4. Click Save Configuration.
    The cron run time interval is updated.
