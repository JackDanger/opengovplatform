Resetting Themes

If the new theme configuration settings generate errors or produce unintended results on the OGPL Web site, you can reset all the settings of the current theme to their default values. When you restore the theme settings to their default values, the configuration settings that you specified in the Global settings tab are used. The Global settings options control the default display settings for the entire OGPL site, across all themes, unless you override them using a specific theme. For more information, see Configuring Global Settings for Themes.

To reset the theme configuration to default values:

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Themes.
    The Themes page appears with a list of the all the themes currently available in the OGPL Control Panel.

  2. Click Reset to defaults.
    The current theme settings are reset to the values that you specified in the Global Settings tab.
