Setting the Action Status for a Feedback

You can set the action status for a feedback to specify the next activity that must be performed on the feedback. The VRM Administrator decides the options that are available in the Action Status list.

To set an action status for a feedback:

  1. On the navigation menu, click the VRM link.
    The VRM page appears.

  2. Optional: You can filter the feedback list. For more information, see Filtering the list of Feedback.

  3. In the row of the feedback for which you want to select the action status, click the Edit this record icon .
    The Feedback Details Edit page appears.

  4. In the Action Status list, click the appropriate action.
    Example: If a request made in this feedback has already been fulfilled, you can click Already Published.

  5. Click Submit.
    The feedback saves with the action status.
