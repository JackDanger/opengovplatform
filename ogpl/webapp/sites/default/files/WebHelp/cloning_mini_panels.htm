Cloning Mini Panels

To create a copy of an existing Mini Panel, you can clone that Mini Panel by performing the following steps:

  1. In the Drupal menu, point to Site building, Mini panels, and then click List.
    The Mini Panels page appears.
    The List tab displays all the Mini Panels that are configured in the OGPL Web site.

  2. In the row of the Mini panel that you want to clone, click Clone.
    The Clone Mini Panel page appears.

  3. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Clone Mini Panel page

Field

Description

Administrative title

Enter a title for this Mini Panel. This title appears on the OGPL Web site.

Name

Enter a name for this Mini Panel. This name does not appear on the OGPL Web site, and is only used to identify this Mini Panel within the OGPL Control Panel application.

Ensure that the name:

  • is in lowercase.

  • does not have any punctuation marks, except underscores.

  • is unique.

By default, this box is set to clone_of_<OriginalMiniPanelName>. For example, if you cloned a mini panel named myminipanel, this box displays clone_of_myminipanel.

Administrative description 
(Required)

Enter the description and any additional information for this Mini Panel, such as the purpose of this mini panel.

Category

Enter a category for this Mini Panel. The Mini Panel is grouped according to this category in the Add content pages.

Leave this box blank to group this Mini Panel by the default category Mini panels

 

  1. Optional: If you only want to temporarily update the latest updates that you made to this Mini Panel without saving them to the database, click Update.
    Note: The updates you made to this Mini Panel are not saved to the database until you click Save.

  2. Click Save.

A copy of the Mini Panel that you cloned is saved with the updates, and appears in the List tab.
