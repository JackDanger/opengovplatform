Setting the Contact Us Page View

The Contact Us page displays the office address of the OGPL Web site and the e-mail address through which a visitor can contact OGPL administrators. The visitors can also use the Contact Us form to provide feedback about the Web site. The link to the Contact Us page appears in the footer  of the home page and the inner pages of the OGPL Web site.

You can set the view that must appear when a visitor clicks the Contact Us link. The Contact Us page has the following views:

  1. The contact-us page that displays only the office address and e-mail address.

    The following image illustrates how the contact-us page appears on the OGPL Web site.


  2. The contactus page that displays a feedback form. The visitors can use this form to provide feedback about the OGPL Web site.

    The following image illustrates how the contactus page appears on the OGPL Web site.

To set the view that must appear when a visitor clicks the Contact Us link:

  1. In the navigation menu, point to Configurable Items, and then click Manage Menus.
    The List menus page appears.

  2. If you want to set the page view when the visitor clicks the Contact Us link on the home page, click the Home Page Footer Menu link.
    If you want to set the page view when the visitor clicks the Contact Us link on the inner pages, click the Inner Page Footer Menu link.
    The List items page appears.

  3. Under the Operations column, click the edit link for the Contact Us menu item.
    The Edit Menu Item page appears.

  4. If you want to use the page that displays the office address and the e-mail address, in the Path (required) box, enter contact-us.
    If you want to use the page that displays a feedback form, in the Path (required) box, enter contactus.

  5. Click Save.
    The appropriate Contact Us page appears when a visitor clicks the Contact Us link on the OGPL Web site.
