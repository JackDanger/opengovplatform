Configuring Color Themes

The visitors of the OGPL Web site can apply a specific color theme for the OGPL Web site. By default, the OGPL Web site provides the following color themes:

  • Grey

  • Blue

  • Pink

  • Green

The following image illustrates how color themes appear on the OGPL Web site.

You can add color themes for the OGPL Web site, but you cannot update or delete the color themes that are available by default.
