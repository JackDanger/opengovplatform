Deleting Contrast Options

You can delete a contrast option from the OGPL Control Panel application and OGPL Web site. You cannot restore a deleted contrast option.

Note: You cannot delete the default contrast options, such as High Contrast and Standard Contrast.

To delete a contrast option:

  1. In the navigation menu, point to Configurable Items, Manage Header Content, and then click Manage Theme Files.
    The Theme Files page appears and displays the list of the contrast options that you created.

  2. In the row of the contrast option that you want to delete, click the delete link.
    A confirmation message appears.

  3. Click Delete.
    The contrast option is deleted and its icon disappears from the OGPL Web site.
