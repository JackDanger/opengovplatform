Tracking Revisions of a Content

You can view and track the updates made to a specific content by performing the following steps:

  1. In the navigation menu, click the Home link.

  2. In the row of the content for which you want to view the revision history, click the Preview this record icon .
    The     View page appears.

  3. Click the Track tab.
    A page that displays the list of updates made to the content appears. The following table lists each column and its description.
    Table: Description of columns in the Track page

Column

Description

Time

Displays the date and time when the content was modified.

Referrer

Displays URL of the page from where the user accessed this content.

User

Displays the username of the user who updated this content.

Operations

Click the details link to view information about the updates for this record. You can view the following information:

  • URL of the page where this content appears.

  • Title of the content that was updated.

  • URL of the page from where the user accessed this content.

  • Date and time when the content was modified.

  • Username of the user who updated the content.

  • IP address of the machine that was used to access the content record.
