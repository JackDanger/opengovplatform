CMS Administrator

The CMS Administrator manages the complete content management workflow, from creating users and adding user roles to CMS workflow, configuring various Web site options required for the OGPL Web site, to generating reports. The CMS Administrator can perform the following tasks:

 

  • Add, edit or delete a user account. For more information, see Managing Users.

  • View or edit the currently configured agencies. For more information, see Managing Agencies.

  • View the themes currently available for the OGPL Web site. For more information, see Viewing the List of Themes.

  • Enable a theme for a user role. For more information, see Enabling a Theme.

  • Specify a default theme for the OGPL Web site. For more information, see Selecting a Default Theme.

  • Configure the global settings for all the themes. For more information, see Configuring Global Settings for Theme.

  • Manage the Blocks content. For more information, see Managing Blocks Content.

  • Configure the OGPL Web site parameters. For more information, see Configuring the Site Information.

  • Create or edit content to be published to the OGPL Web site. For more information, see Managing Content.

  • Configure the accessibility settings, such as the text resize feature. For more information, see Managing Accessibility.

  • Configure the search parameters across the entire OGPL Web site. For more information, see Managing Global/Advanced Search.

  • Configure the Blocks to display generic content on the OGPL Web site. For more information, see Managing Blocks Content.

  • Configure the menus and sub-menus to display on the OGPL Web site. For more information, see Managing Menus.

  • Configure the Captcha settings for various OGPL Web pages. For more information, see Managing Captcha.

  • Configure the Homepage Tabs. For more information, see Managing Homepage Tabs.

  • Configure the RSS feed settings for the OGPL Web site. For more information, see Managing RSS Feeds.

  • View CMS Reports, such as Error logs and Access Statistics. For more information see, CMS Reports.

  • Upload Reference Documents. For more information, see Uploading Reference Documents.

  • Contact the Super Administrator of OGPL Control Panel application. For more information, see Contacting the Super Administrator.
