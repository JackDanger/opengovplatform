Blocking Users

When you block a user, that user is unable to log in to the OGPL Control Panel application.

You can block a user by performing the following steps:

  1. In the Drupal menu, point to User Management, Users, and then click List.
    The Users page appears.

  2. In the Users page, under the Operations column, click the Edit link for the user name that you want to block.
    A page that displays the user account information appears.

  3. Scroll down until you see the Status box, and then click Block.

  4. Click Save.
    The user is blocked from accessing the OGPL Control Panel application.
