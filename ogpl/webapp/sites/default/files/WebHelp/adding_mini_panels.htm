Adding Mini Panels

You can add a new Mini Panel and specify its layout.  You can also save the Mini Panel layout as a template and use the template when you create other Mini Panels.

To add a Mini Panel:

  1. In the Drupal menu, point to Site building, Mini panels, and then click Add.
    The Add a New Mini Panel page appears.

  2. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Add a New Mini Panel page

Field

Description

Administrative title

Enter a title for this Mini Panel. This title appears on the OGPL Web site.

Name

Enter a name for this Mini Panel. This name does not appear on the OGPL Web site, and it is only used to identify this Mini Panel within the OGPL Control Panel application.

Ensure that the name:

  • is in lowercase.

  • does not have any punctuation marks, except underscores.

  • is unique.

Administrative description 
(Required)

Enter the description and any additional information for this Mini Panel, such as the purpose of this mini panel.

Category

Enter a category for this Mini Panel. The Mini Panel is grouped according to this category in the Add content pages.

Leave this box blank to group this Mini Panel by the default category Mini panels.

 

  1. Click Continue.
    The Add A New Mini Panel page displays the context settings.

  2. In the Required Context list, click the appropriate option.

  3. Click Add required context.
    The Add required context page appears.

  4. Optional: Update the Identifier and Keyword boxes.

  5. Click Save.
    In the Contexts list, click the appropriate option.

  6. Click Add context.
    The Add required context page appears.

  7. If required, update the Identifier and Keyword boxes.

  8. In the Enter the title or NID of the node box, enter the first few characters of the node that you want to add.
    The list of available nodes appears.

  9. Click the appropriate node.

  10. Click Save.
    The Add required context page appears.

  11. Click Continue.
    The Add A New Mini Panel page displays the content layout options.

  12. In the Category list, click the layout category that you want to use for this Mini Panel.
    The Add A New Mini Panel page displays various layouts available for the category that you selected.

  13. In the Add A New Mini Panel page, click the layout that you want to use for this Mini Panel.

  14. Click Continue.
    The Add A New Mini Panel page displays the display settings.

  15. Click the appropriate Title type for the title of this Mini Panel.

If you want to specify a particular title for this Mini Panel, click Manually set, and then enter the title that you want to use in the Title box.

If you do not want to use a title for this Mini Panel, click No title.

If you want to use a title from a pane, click From pane.

  1. To manually adjust the Mini Panel layout, click Show layout designer.

  2. To save this layout for future use, click Reuse layout.
    The Save this layout for reuse page appears.

  3. Enter the appropriate information in each field.

  4. Click Save.

  5. To view the layout of this Mini Panel, click Preview.
    The Live preview section displays a preview of this Mini Panel as it will appear on the OGPL Web site.

  6. Click Finish.
    The Mini Panel is created and it appears in the List tab.
