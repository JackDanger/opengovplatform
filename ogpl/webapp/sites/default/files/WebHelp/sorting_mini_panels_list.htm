Sorting Mini Panels List

Sorting the list of Mini Panels helps you quickly analyze, visualize, and understand the currently configured Mini Panel on the OGPL Web site. By default, the list of Mini Panels is sorted according to the Title column. You can also sort the list according to various criteria, such as the Mini Panel name, category, layout or storage.

 

To sort the list of Mini Panels:

  1. In the Drupal menu, point to Site building, Mini panels, and then click List. The Mini Panels page appears. The List tab displays all the Mini Panels that you configured in the OGPL Web site.

  2. In the Sorted by list, click the column by which you want to sort the list of Mini Panels. The list of Mini Panels appears according to the sort criteria that you specified.

  3. Optional: To sort the list of Mini Panels in ascending order of the selected column, in the Order list, click Up.
    To sort the list of Mini Panels in descending order of the selected column, in the Order list, click Down.
    To sort in ascending order of the Title column, click Reset.
