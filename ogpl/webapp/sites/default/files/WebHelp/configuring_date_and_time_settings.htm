Configuring Date and Time Settings

You can configure the date and time settings for the OGPL Web site by performing the following steps:

  1. In the Drupal menu, point to Site configuration, and click Date and time.
    The Date And Time page appears.

  2. In the Locale settings section, enter the appropriate information. The following table lists each field and its description:

Table: Description of options in the Locale settings section

Title

Description

Default time zone

Click the appropriate region that has the time zone that you want to set as the default time zone for the OGPL Web site.

User-configurable time zones

You can specify if you want to specify your own time zone by clicking the appropriate option.

By default, the option is set to Enabled.

First day of week

Click the appropriate day that you want to set as the first day of the week.

Use ISO-8601 week numbers

Select this check box if you want to use the ISO 8601-compliant representation of date and time.
Note: If this check box is selected, you must set Monday as the first day of the week.

 

  1. Click Save configuration.

The date and time settings are updated on the OGPL Web site.
