Editing Users

You can update the information about a user, such as the username, password, roles, by performing the following steps:

  1. In the Drupal menu, point to User Management, Users, and then click List.
    The Users page appears.

  2. In the Users page, under the Operations column, click the Edit link for the user that you want to update.
    A page that displays the user account information appears.

  3. Update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Users page

Title

Description

Username

Update the user name of this user.

All instances of this username in the OGPL Control Panel application are updated.

E-mail address

Update the e-mail address of this user.

All instances of this e-mail address in the OGPL Control Panel application are updated.

Password

If you want to assign a new password to this user, enter a password.

Confirm Password

Enter the same password that you entered in the Password box.

Status

By default, this is set to Active.

When the Status is set to active, the user can log in to and access the areas of the OGPL Control Panel application that are designated to the role that this user is assigned to.

If you want to block this user from logging in to the OGPL Control Panel application, click Blocked.

Roles

Select the appropriate check box to update the roles for the user.

Note: You can assign more than one role to a user.

Force password change on next logon

If you want the user to change the password when the user logs on to the OGPL Control Panel application, select this check box.

Theme Configuration

Update the theme of the application that user must have access to.

Default time zone

Click the appropriate time zone for the location of this user.
  1. Click Save.
    The information for this user is updated in the OGPL Control Panel application.
