Importing Mini Panels

You can reuse and import an existing Mini Panel to create a new Mini Panel on the OGPL Web site. To import a Mini Panel, you must have its export definition.

To import a Mini Panel:

  1. Copy the export definition of the Mini Panel that you want to reuse. For more information, see Exporting Mini Panels.

  2. In the Drupal menu, point to Site building, Mini panels, and then click Add.
    The Import Mini Panel page appears.

  3. In the Mini panel code box, paste the export definition of the Mini Panel that you want to import.

  4. Optional: If you want to overwrite the existing Mini Panel with the one that you are importing, select the Allow import to overwrite an existing record check box.

  5. Click Continue.
    The Import Mini Panel page displays the title, name, category and other details for the Mini Panel.

  6. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Import Mini Panel page

Field

Description

Administrative title

Enter a title for this Mini Panel. This title appears on the OGPL Web site.

Name

Enter a name for this Mini Panel. This name does not appear on the OGPL Web site, and is only used to identify this Mini Panel within the OGPL Control Panel application.

Ensure that the name:

  • is in lowercase.

  • does not have any punctuation marks, except underscores.

  • is unique.

Administrative description 
(Required)

Enter the description and any additional information for this Mini Panel, such as the purpose of this mini panel.

Category

Enter a category for this Mini Panel. The Mini Panel is grouped according to this category in the Add content pages.

Leave this box blank to group this Mini Panel by the default category Mini panels

 

  1. Optional: If you only want to temporarily update the latest updates that you made to this Mini Panel without saving them to the database, click Update.
    Note: The updates you made to this Mini Panel are not saved to the database until you click Save.

  2. Click Save.

The Mini Panel that you imported is saved with the updates that you made.
