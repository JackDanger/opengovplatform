Adding a Feedback Review

When a feedback is assigned to you, as a feedback owner, you must analyze the feedback, and add a feedback review about the steps taken to address the feedback.

Note: You can add a feedback review only if the visitor has not provided an e-mail address while submitting the feedback.

To add a feedback review:

  1. In the navigation menu, click the Home link.
    The My List page appears.

  2. Optional: You can filter the feedback list. For more information, see Filtering the list of Feedback.

  3. In the row of the feedback that you want to analyze, click the Edit this record icon.
    The Feedback Details Edit page appears.

  4. View or update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Feedback Details Edits page

Title

Description

Sender Name

Displays the name of the person who provided this feedback. If the visitor has not provided a name while submitting the feedback, this field appears blank.

Subject

Displays the subject of this feedback. If the visitor has not provided a subject while submitting the feedback, this field appears blank.

Feedback

Displays the feedback.

Action Status

Select the appropriate action that the feedback owner must perform.

  1. Click Review.
    The Create Feedback Review page appears.

  2. The Reviewed by box displays your username.

  3. In the Comments box, type the feedback review.

  4. Click Submit.
    The feedback review is saved and appears in the Reply / Review tab on the Feedback Details Edit page.

The VRM Administrator analyzes this feedback review to verify whether you have taken the appropriate steps to address this feedback.
