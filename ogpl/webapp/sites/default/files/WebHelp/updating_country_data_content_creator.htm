Updating Country Data

The header section of the OGPL home page contains flags that link to the Open Data Site of other countries. The Country Data feature allows you to add information about each country, such as the flag image and the dataset link. When you create the country data, the country information also appears in the WW Data Sites page, which displays all the Open Data Sites for every country. You can also upload the image of the flag that must appear as the Site Header image at the top left area of the OGPL Web site.

You must update the country data with the appropriate content and send it to the Moderator for review and approval.

The following image illustrates how Country Data appears on the OGPL Web site.

To update the content for a Country Data:

  1. In the navigation menu, click the Home link.

  2. In the Content List section, click the Edit this record icon   for the Country Data that you want to update.
    The Edit page appears.

  3. Update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Edit page

Field

Description

Title
(Required)

Update the name of the country for which you are creating this Country Data link. The name of the country appears as the ScreenTip for the flag image. This information also appears in the WW Datasite page.

Site Header Image
(Required)

Upload the image of the flag that must appear at the top left of the header of the OGPL Web site.

Note: The recommended image dimension is 38x24 pixels.

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Site Header Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

Flag Image

Upload the image of the flag that must appear at the top right of the header on the OGPL Web site. The recommend image dimension is 22x23 pixels.

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Flag Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

WW Data Site Image
(Required)

Upload the image of the flag that must appear in the WW Data Sites page of the OGPL Web site. The recommend image dimension is 157x105 pixels.

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Open Data Site Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

Union Government Name
(Required)

Update the Government name for this country. This name appears on the OGPL Web site. This information also appears as the ScreenTip on the WW Datasite page.

National/Country Portal

Update the main portal URL of the country for which you are creating this Country Data content.

Dataset Link

Update the dataset link URL of the country for which you are creating this Country Data content.

Latitude
(Required)

Update the latitude coordinates of the country for which you are creating this Country Data content.

Longitude
(Required)

Update the longitude coordinates of the country for which you are creating this Country Data content.

State/City/Agency Data

Region Type

Click the appropriate region information, such as State, City, Agency, Municipality and Sub-agency, this country.

This information appears when a visitor clicks the name of this country in the WW Datasite page.

State/Agency

Update the name of the state or agency of the selected region. This is a mandatory field if you add a City/sub-agency/Municipality.

This information appears when a visitor clicks the name of this country in the WW Datasite page.

Note: Once you update the State/Agency name and save this country data, the State/Agency name appears as an autocomplete option when you update this country data and specify a new State/Agency.

City/Sub-agency

Update the name of the city, sub-agency, or municipality of the selected region.

This information appears when a visitor clicks the name of this country in the WW Datasite page.

Note: This field is available only when you select City, Municipality or Sub-Agency as the Region Type. This field cannot be left blank.

Portal URL

Update the main portal link URL of the selected region.

Dataset URL

Update the dataset URL that belongs to the selected region.

Latitude

Update the latitude coordinates for the selected region.

Longitude

Update the longitude coordinates for the selected region.

Content Management

If you want to send the content to the Moderator, click Moderator Review.

If you want to retain the content with you for further updates, click Create/Edit Content.

Comment

Update additional information about this content. This information appears while viewing the workflow history for this content.

Revision Information - Log Message

Update additional information about the updates that you made to this Country Data. This information appears while viewing the revision history for this content.

Enter the Expiry Date

Update the date when the information of this Country Data is outdated.

Note: The content remains on the OGPL Web site until it is unpublished or deleted by the Super Administrator.

Notify Before (Days)

Update the number of days before the expiry date when the Content Creator must receive a notification about the expiry of the content.

 

    1. Click Save. The content record is updated.

Note: If you want to see how the content looks on the OGPL Web site before the content is published, click Preview.
