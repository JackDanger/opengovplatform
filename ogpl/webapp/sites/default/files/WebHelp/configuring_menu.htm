Configuring Menus

You can specify an existing OGPL Web site menu as the default menu in the Menu settings section that appears when creating the content for the OGPL Web site.  For more details, see any topic for creating the content on the OGPL Web site.

You can also specify the sources that you want to use by default, for primary links and secondary links. Primary links are the menus that are visible on the OGPL Web site. Secondary links are the sub-menus that appear when a visitor clicks a primary link.

To configure the menu settings:

  1. In the navigation menu, point to Configurable Items, and then click Manage Menu.
    The Menus page appears.

  2. Click the Settings tab.
    The Settings tab displays the menu settings that you want to configure.

  3. Select the appropriate menu settings. The following table lists each field and its description.
    Table: Description of menus in the Settings tab

Field

Description

Default menu for content

Click the menu that you want to use as the default menu for the content on the OGPL Web site. The default option is Primary links.

For example, if you want to use the footer section on the home page of OGPL Web site as the default menu, click Home Page Footer Menu.

Source for the primary links

Click the menu that you want to use as the primary link. The default option is Primary links.

For example, if you want to use Global Navigation as the primary link, click Global Navigation.

Source for the primary links

Click the menu that you want to use as the secondary link. The default option is Secondary links.

For example, if you want to use Features as the secondary menu, click Features.

Note: You can use the same menu for primary links and secondary links. In such a case, the sub-menus of the primary links will appear as the secondary links.

 

  1. Click Save configuration.
    The menu settings are saved in the OGPL Control Panel application.
