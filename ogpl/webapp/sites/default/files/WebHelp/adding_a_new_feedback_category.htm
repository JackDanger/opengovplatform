Adding a Feedback Category

If you cannot categorize a feedback under any existing feedback category, you can add a new feedback category. While adding a feedback category, you can also add a pre-defined text message. When you add the pre-defined text to a category, the category name also appears in the Pre-Formatted Response list on the Create Feedback Reply page. The feedback owner can select the category name from the Pre-Formatted Response list and use the pre-defined text while replying to a feedback.

To add a feedback category:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Manage Categories tab.
    The Category List page appears.

  3. Click the Add tab.
    The Add Category page appears. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Add Category page

Field

Description

Category Name
(Required)

Enter the name for this feedback category. For example, if you want to create a category for all the feedback related to the content on the OGPL Web site, enter Content Related.

Pre-defined Text

Enter the default text that a feedback owner can use in the body of the feedback reply e-mail for this category.

The feedback owner can use the pre-defined text by selecting the name of the category from the Pre-Formatted Response list on the Create Feedback Reply page.

Contact Category

If you want this category to appear in the Category list of the Contact Us page form on the OGPL Web site, select this check box.

  1. Click Add.
    This category is added to the Category List. You can now categorize feedback under this category.
