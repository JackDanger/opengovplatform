Disabling Mini Panels

You can disable a Mini Panel to hide it from the OGPL Web site. A disabled Mini Panel is not deleted from the database and it remains in the OGPL Control Panel.

To disable a Mini Panel:

  1. In the Drupal menu, point to Site building, Mini panels, and then click List.
    The Mini Panels page appears.
    The List tab displays all the Mini Panels that are configured in the OGPL Web site.  

  2. In the row of the Mini Panel that you want to disable, click Disable.
    The Mini Panel is disabled and it disappears from the OGPL Web site.
