Tracking the Delay Time

You can view and track the updates made to the default delay time for a feedback. The Delay Time History page displays the details of the updates made to the delay time.

To track the changes made to the delay time:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Manage Delay Time tab.
    The Manage Delay Time page appears.

  3. Click the History tab.
    The Delay Time History page appears with the list of all the updates made to the delay time. The following table lists each column and its description.

Table: Description of Columns in the Delay Time History page

Column

Description

Time

Displays the date and time when the VRM Administrator updated the delay time.

Details

Displays details of the delay time update, such as the user who performed this update and the update made to the delay time.
