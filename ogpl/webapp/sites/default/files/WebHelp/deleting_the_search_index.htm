Deleting the Search Index

If the index gets corrupt or you have installed a new schema.xml for the search index, you can delete the existing index and create a new search index.

To delete the search index:

  1. In the navigation menu, point to Configurable Items, Manage Header Content, and then click Manage Global and Advance Search.
    The Settings page appears.

  2. Click the Search Index tab.
    The Apache Solr page appears.

  3. Click Delete the index.
    A page that displays the confirmation message appears.

  4. Click Delete.
    The search index is deleted from the index file created by the Apache Solr server to store the index. You can now create a new index by running a cron job.
