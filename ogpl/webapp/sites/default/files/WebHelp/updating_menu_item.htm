Updating a Menu Item

You can update a menu item to specify a new link for a page or a new title for the menu item by performing the following steps:

  1. In the navigation menu, point to Configurable Items, and then click Manage Menu.
    The List menus page appears.

  2. Click the primary link to which the menu item that you want to update belongs. For example, if you want to update a menu item that belongs to the footer section of the Home page, click the Home Page Footer Menu link.
    The List items page appears.

  3. In the Operations column, click the edit link for the menu item that you want to update.
    The Edit Menu Item page appears. Update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Edit Menu Item page

Field

Description

Path
(Required)

Enter the new URL for the page that you want to link with this menu item.

Menu Link Title
(Required)

Enter the new title that must appear as the link on the OGPL Web site.

Description

Update the text that must appear as the ToolTip for this menu item.

 

Note: This is an optional field.

Enabled

If you want this menu item to appear on the OGPL Web site, select this check box.

If you do not want this menu item to appear on the OGPL Web site, clear this check box.

Expanded

If you want this menu item to appear expanded on the OGPL Web site, select this check box.

If you do not want this menu item to appear expanded on the OGPL Web site, clear this check box.

  1. Click Save.
    This menu item is updated.
