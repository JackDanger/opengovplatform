Sending the Content Back to the Content Creator

If the content, which the Content Creator sent for review, is not appropriate, you can send the content back to the Content Creator.

While sending the content back to the Content Creator, you can also view the workflow history for that content. The Workflow History section displays details, such as the current workflow state for this content, when the content was created, when you send the content back to the Content Creator, when the content was sent to a Publisher for approval, or how many times the content has gone back for updates.

To send the content back to the Content Creator:

  1. In the navigation menu, click the Home link.

  2. In the Content List section, click the Send this record back in workflow icon for the content that you want to send back to the Content Creator.
    A page that displays the title of the content that you selected appears.

  3. In the Change Content Management state list, click Create/Edit Content.

  4. In the Comment box, enter your comments, such as the reason for sending the content record back to the Content Creator or the updates required.

  5. Click Submit.
    The content is sent back to the Content Creator.
