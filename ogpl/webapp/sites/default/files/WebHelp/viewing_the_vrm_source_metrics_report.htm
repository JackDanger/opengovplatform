Viewing the VRM Source Metrics Report

The VRM Source Metrics report displays the percentage of feedback that is received from a specific feedback source page, such as the Contact Us page, Feedback page, Ratings page, or Suggest Dataset page on the OGPL Web site. This report enables you to analyze the pattern of feedback received from every source. For example, if the Contact Us page is generating the least amount of feedback, this might mean that the Contact Us link is not easily accessible to visitors.

To view the VRM Source Metrics report, in the navigation menu, point to Reports, VRM Agency Reports, and then click VRM Source Metrics.

Tip: To know the number of feedback for each source, place the mouse pointer over the appropriate source on the pie chart.

You can also view the VRM Source Metrics report for a specific year, month, action status, and feedback category by performing the following steps:

  1. In the Select Year list, click the year for which you want to analyze the feedback source.

  2. In the Select Month list, click the month for which you want to analyze the feedback source.

  3. In the Select Action Status list, click the action status for which you want to analyze the feedback source.

  4. In the Select Category list, click the feedback category for which you want to analyze the feedback source.

  5. Click Submit.
    The VRM Source Metrics report for the selected criteria appears.
