Configuring the Date and Time Format

Various pages of the OGPL Web site use different formats to display date and time. For example, the Agency Wise page displays the date and time at which a dataset record was submitted. Using the Date and Time page or Views, you can configure the format of the date and time that must appear on various pages of the OGPL Web site. The following table lists the pages of the OGPL Web site that display the date and time along with and their date and time format source.
