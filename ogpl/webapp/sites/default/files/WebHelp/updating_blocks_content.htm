Updating Blocks Content

You can edit the content of a block displayed on the OGPL Web site by performing the following steps:

  1. In the navigation menu, point to Configurable Items, and then click Manage Blocks Content.
    The     Manage Blocks Content page appears.

  2. In the row of the block for which you want to update the content, click the edit link.
    The     Edit tab displays the settings for block that you selected.

  3. Update the appropriate block content. For more information, see Updating Blocks.
