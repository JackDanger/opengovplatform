Creating Blocks

A block is a box of content that is rendered into a region of the OGPL Web site.

Whenever new content is required, you must create a block, and assign the block to a Content Creator and a Moderator. The Content Creator updates the block by adding the required content, and the Moderator reviews the content to verify whether the content is appropriate for the OGPL Web site.

The following image illustrates how a Block appears on the OGPL Web site.

To create a block:

  1. In the navigation menu, point to Configurable Items, and then click Create Content.

  2. The Create Content page appears.

  3. Click the Block link.
    The Create Block page appears.

  4. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Create Block page

Title

Description

Title

(Required)

Enter an appropriate title for this block.

Instructions

Enter the guidelines that the assigned Content Creator must follow while updating the content for this block. The Moderator and the Publisher reviews the content based on these guidelines.

Menu settings

For more information about Menu settings, click here.

Body

The assigned Content Creator adds or updates the content that must appear on this block.

Note: You can also provide this content.

Input format

Click the format in which the Content Creator can update content in the Body field. For example: Filtered HTML, Full HTML, PHP Code, or Plain Text.

Revision information -

Log message

Enter the reason for creating this block.

URL path settings

By default, the Automatic alias check box is selected. The application automatically assigns the URL for this block.

If you want to assign a specific URL for this block, perform the following steps:

  1. Click to clear the Automatic alias check box.

  2. In the URL path settings box, type the URL that you want to assign for this block.

Select A Content Creator

(Required)

Click the username of the Content Creator who must update the content for this block.

Select A Moderator

(Required)

Click the username of the Moderator who must review the content.

Enter the Expiry Date

Enter the date when the information in this block is outdated.

The content remains on the OGPL Web site until it is unpublished by the Super Administrator.

Notify Before (Days)

Enter the number of days before the expiry date when you want to receive a notification about the expiry of the content.
  1. Click Save.

An entry for this block appears in the content list of the assigned Content Creator and Moderator.

Note: You do not need to enter values for all the fields on this page. You can enter values for the mandatory fields only, and assign the block to a Content Creator. You or a Super Administrator can enter appropriate values in the remaining fields after the content is created, reviewed, and published.

Table: Description of fields in the menu setting section

Title

Description

Menu link title

Enter the title of the menu item for this block. This menu item appears within the main menu on the OGPL Web site.

Tip: The title of this menu item can be the same as the page title.

Note: If you do not want to add this new block to a menu, leave this field blank.

Parent item

Click the menu under which the menu link for this block must appear. For example, if you want the menu link for this block to appear in the navigation bar of the OGPL Web site, click Global Navigation.

Weight

Menu links that have lesser weight appear closer to the parent item.

Assign the appropriate weight for the menu link of this block depending on where you want the link to appear. For example, if you selected Global Navigation as the Parent item, and you want the menu link of this block to appear on the navigation bar, enter 0.
