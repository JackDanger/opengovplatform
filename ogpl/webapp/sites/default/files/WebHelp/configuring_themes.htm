Configuring Themes

You can customize a theme by configuring the logo settings and shortcut icon settings. You can also specify the page elements that you want to display in the theme.

To configure a theme:

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Themes.
    The Themes page appears with a list of the all the themes currently available in the OGPL Control Panel.

  2. In the row of the theme that you want to configure, click the configure link.
    The Configure tab displays the settings for the theme that you selected.

  3. Configure the appropriate theme settings. The following table lists each field and its description.

Table: Description of fields in the Configure tab on Themes page

Field

Description

Toggle Display

Select the check box for each page element that you want to display on the OGPL Web site.

Click to clear the check box for each page element that you do not want to display on the OGPL Web site.

For example, if you want to display a logo on the OGPL Web site, select the Logo check box.

Logo image settings

This section enables you to specify a logo for the OGPL Web site.

Use the default logo

If you want to use the default logo on the OGPL site, select this check box.

Path to custom logo

If you want to another logo instead of the default logo, enter the path to the image that you want to use.

Upload logo image

Upload the image that you want to use as logo for the OGPL Web site.

To upload an image:

  1. Click Browse. The File Upload dialog box appears.

  2. Browse for and select the image that you want to upload.

  3. Click Open.

  4. Click Upload. The image is uploaded and a preview of the image is displayed.

 

Shortcut icon settings

This section enables you to specify a shortcut icon to display for OGPL Web site URL.

Use the default shortcut icon

If you want to use the default shortcut icon for the OGPL Web site URL, select this check box.

Path to custom icon

Enter the path to the image that you want to use as your shortcut icon file instead of the default shortcut icon.

Upload icon image

If you do not have direct file access to the OGPL Web server, you can upload your shortcut icon on the Web server.

Upload the image that you want to use as shortcut icon for the OGPL Web site URL.

To upload an image:

  1. Click Browse. The File Upload dialog box appears.

  2. Browse for and select the image that you want to upload.

  3. Click Open.

  4. Click Upload. The image is uploaded and a preview of the image is displayed.
