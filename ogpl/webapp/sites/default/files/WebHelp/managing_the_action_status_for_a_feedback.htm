Managing the Action Status for a Feedback

The action status provides information about the feasibility of the feedback implementation. For example, a visitor requests for a dataset and you think that it is possible to publish the requested dataset on the OGPL Web site, you can update the Action Status for this feedback to Actionable. However, if you think that is not possible to publish the requested dataset on the OGPL Web site, you can update the Action Status for this feedback to Not Actionable. You can also add, edit, delete, and track the action status.

Note: It is not mandatory to specify an action status for a feedback while assigning the feedback.

The action status information is also used for generating charts in the VRM Action Metrics report. For more information, see VRM Action Metrics Report.
