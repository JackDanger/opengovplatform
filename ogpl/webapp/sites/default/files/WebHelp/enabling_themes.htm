Enabling Themes

You can enable one or more themes so that you can use or apply these themes for the OGPL Control Panel application.

To enable a theme:   

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Themes.
    The Themes page appears with a list of the all the themes currently available in the OGPL Control Panel.

  2. In the row of the theme that you want to enable, select the check box in the Enabled column.
    Note: You can enable multiple themes.

  3. Click Save configuration.
    The theme that you selected is enabled and is available for use in the OGPL Control Panel application.
