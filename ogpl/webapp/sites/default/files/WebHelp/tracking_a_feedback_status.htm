Tracking Feedback Action Status

You can view and track the updates for a feedback action status. The VRM Workflow Status History page displays the time and details of each update made to the feedback action status.

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Status tab.
    The VRM Workflow Status page appears.

  3. Click the History tab.
    The VRM Workflow Status History page appears with the list of updates made to each feedback action status. The following table lists each column and its description.

Table: Description of Columns in the VRM Workflow Status  History page

Column

Description

Time

Displays the date and time when the VRM Administrator modified the feedback action status.

Details

Displays details of the feedback action status update, such as the user who performed this update and the update made to the feedback action status description.
