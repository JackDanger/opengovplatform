Updating Mini Panels

You can update an existing Mini panel by modifying its title, description or category. You cannot update the name of an existing Mini Panel.

To update a Mini Panel:

  1. In the Drupal menu, point to Site building, Mini panels, and then click List.
    The Mini Panels page appears.
    The List tab displays all the Mini Panels that are configured in the OGPL Web site.  

  2. In the row of the Mini panel that you want to update, click Edit.
    The Edit Mini Panel page appears.

  3. Update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Edit Mini Panel page

Field

Description

Administrative title

Update the  title of  this Mini Panel. This title appears on the OGPL Web site.

Name

Update the name of this Mini Panel. This name does not appear on the OGPL Web site, and is only used to identify this Mini Panel within the OGPL Control Panel application.

Ensure that the name:

  •  is in lowercase.

  • does not have any punctuation marks, except underscores.

  • is unique.

Administrative description 
(Required)

Update the description and any additional information for this Mini Panel, such as the purpose of this mini panel.

Category

Update the  category of  this Mini Panel. The Mini Panel is grouped according to this category in the Add content pages.

Leave this box blank to group this Mini Panel by the default category Mini panels.

 

  1. Click Save.
    The Mini Panel is saved with the latest updates that you made.
