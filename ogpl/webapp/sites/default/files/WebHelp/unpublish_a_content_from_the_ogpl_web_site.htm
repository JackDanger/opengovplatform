Unpublishing Content from the OGPL Web site

If you do not want the visitors of the OGPL Web site to view a content record, you can unpublish that content from the OGPL Web site. When you unpublish a content record, it is only removed from the OGPL Web site and it is not deleted from the database, which means that you can reuse the content later.

To unpublish a content record from the Web site:

  1. In the Drupal menu, point to Content Management, Content and then click List.
    The Content page appears.

    You can view the list of the content and details on the Content page, and then decide the content that you do not want to display on the OGPL Web site. The following table lists each column and its description.

Table: Description of columns in the Content page

Columns

Description

Title

Displays the title of the content record.

Type

Displays the content type, such as Page, Block, FAQ, Feature Gallery, and so on.

Author

Displays the username of the user who created the content.

Status

Displays the current status of the content record. For example, if the content record is currently published on the OGPL Web site, the status appears as Published.

Language

Displays the language of the content record.

Operations

Allows you to edit the content record.

  1. Under the Title column, select the check box for each content record that you want to unpublish from the OGPL Web site.

  2. In the Update options list, click Unpublish.

  3. Click Update.
    The content that you selected is removed from the OGPL Web site and the status of the content is updated to not published.
