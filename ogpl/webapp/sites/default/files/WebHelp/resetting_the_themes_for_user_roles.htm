Resetting the Themes for User Roles

When the updated themes settings result in an error or show unexpected results, you can reset all the themes configuration settings to their default values.

To reset the themes configuration to their default values:

  1. In the Drupal menu, point to User management, and then click Role theme switcher.
    The Role Theme Switcher page appears.

  2. Scroll down until you reach the end of the page, and then click Reset to defaults.
    The default theme is applied to each user role.
