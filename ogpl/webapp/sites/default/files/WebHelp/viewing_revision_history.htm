Viewing Revision History

You can view the revision history of a content record and track the updates between multiple versions of a specific content.

To view the revision history of a content record:

  1. In the navigation menu, click the Home link.

  2. In the row of the content for which you want to view the revision history, click the Preview this record icon .
    The View latest page appears.

  3.  Click the Revisions tab.
    A page that displays the revisions history appears.

  4. To view a previous version of the content, click the date for the version that you want to view.
    The appropriate version of the content appears.

  5. If you want to update the previous version of the content, click the Edit tab.
    A page that displays the title of the content that you selected appears. For more information on how to update content, see <Editing the Content>
