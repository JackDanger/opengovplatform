Resetting Captcha

When the updated CAPTCHA settings results in an error or show undesirable results on OGPL Web site, you can reset all the CAPTCHA settings for the particular CAPTCHA type to their default values.

Note: Resetting the CAPTCHA settings to their default values will remove all the latest changes that you made.

To reset the CAPTCHA settings to their default values:

  1. In the navigation menu, point to Configurable Items, and then click Manage Captcha.
    The CAPTCHA page appears.

  2. Click the tab for the CAPTCHA type, Image CAPTCHA or reCAPTCHA, that you want to reset to its default values.
    The CAPTCHA tab that you selected displays the current settings for the CAPTCHA.

  3. Scroll down until you reach the end of the page, and then click Reset to defaults.
    The CAPTCHA settings are reset to their default values.
