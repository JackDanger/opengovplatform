Deleting a QT block and Quicktab

If you want to remove all the Homepage Tabs from the home page of the OGPL Web site, you can delete the complete Quicktab block. When you delete a Quicktab block, all the details such as, the Homepage Tabs within that Quicktab block are also deleted from the database.

 

To delete a Quicktab block:

  1. In the navigation menu, point to Configurable Items, and then click Manage Homepage Tabs.
    The Edit page appears.

  2. Click the Delete tab.
    The page that displays a confirmation message appears.

  3. Click Delete.
    The Quicktab block and the associated Homepage Tabs are removed from the home page of the OGPL Web site.
