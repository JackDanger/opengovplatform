Creating FAQs

You can create a Frequently Asked Question (FAQ), and assign it to a Content Creator and a Moderator. The Content Creator updates the FAQ by adding the required content, and the Moderator reviews the content to verify whether the content is appropriate for the OGPL Web site. When the FAQ is published, it appears in the FAQ section of the OGPL Web site.

The following image illustrates how an FAQ appears on the OGPL Web site.

To create an FAQ content type:

  1. In the navigation menu, point to Configurable Items, and then click Create Content.

  2. The Create Content page appears.

  3. Click the FAQ link.
    The Create FAQ page appears.

  4. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Create FAQ page

Title

Description

Question

(Required)

Enter the question that this FAQ page must address.

Note: This question is added in the FAQ page on the OGPL Web site.

Menu settings

You do not need to provide information for fields within this section.

Answer

The assigned Content Creator adds or updates the content that must appear on this FAQ page.

Note: You can also provide this content.

Input format

Click the format in which the Content Creator can update content in the Answer field. For example: Filtered HTML, Full HTML, PHP Code, or Plain Text.

Question details

(optional)

Enter details about this FAQ.

Revision information -

Log message

Enter the reason for creating this FAQ.

URL path settings

By default, the Automatic alias check box is selected. The application automatically assigns the URL for this FAQ.

If you want to assign a specific URL for this FAQ, perform the following steps:

  1. Click to clear the Automatic alias check box.

  2. In the URL path settings box, type the URL that you want to assign for this FAQ.

Audio

Upload an audio file or this FAQ file.

Note: You can upload MP3 files.

To upload an audio file:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the file that you want to upload.

  3. Click Open.
    The Audio box displays the path of the file that you selected.

  4. Click Upload.
    The file is uploaded to the database and a link to the audio appears on the page.

  5. Optional: If you want to remove this file, click Remove.
    The uploaded file is removed from the database.

Video

Upload a video file for this FAQ.

Note: You can upload MP4, MPEG, AVI, MPG, WMV, FLV, and MOV files.

To upload a video file:

  1. Click Browse.
    The File Upload page appears.

  2. Click the file that you want to upload.

  3. Click Open.
    The Video box displays the path of the file that you selected

  4. Click Upload.
    The file is uploaded to the database and a link to the video appears on the page.

  5. Optional: If you want to remove the file that you just uploaded, click Remove.
    The uploaded file is removed from the database.

Select A Content Creator

(Required)

Click the username of the Content Creator who must update the content for this FAQ.

Select A Moderator

(Required)

Click the username of the Moderator who must review the content.

Enter the Expiry Date

Enter the date when the information in this FAQ is outdated.

The content remains on the OGPL Web site until it is unpublished by the Super Administrator.

Notify Before (Days)

Enter the number of days before the expiry date when you want to receive a notification about the expiry of the content.
  1. Click Save

An entry for this FAQ appears in the content list of the assigned Content Creator and Moderator.

Note: You do not need to enter values for all the fields on this page. You can enter values for the mandatory fields only, and assign the FAQ page to a Content Creator. You or a Super Administrator can enter appropriate values in the remaining fields after the content is created, reviewed, and published.
