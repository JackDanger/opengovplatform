Deleting a Menu Item

You can delete a menu item when you no longer need to link to a particular page or dataset. When you delete a menu item, all the details for that menu item are removed from the database.

To delete a menu item:

  1. In the navigation menu, point to Configurable Items, and then click Manage Menu.
    The List menus page appears.

  2. Click the primary link to which the menu item that you want to delete belongs. For example, if you want to delete a menu item that belongs to the footer section of the Home page, click the Home Page Footer Menu link.
    The List items page appears.

  3. In the Operations column, click the delete link for the menu item that you want to delete.
    A page that displays a confirmation message appears.

  4. Click Confirm.
    The menu item is deleted from the database and removed from the OGPL Web site.
