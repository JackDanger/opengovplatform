Configuring the Default Delay Time for a Feedback

The default delay time appears when the VRM Administrator assigns a feedback to a feedback owner. The delay time is the time period within which a VRM Administrator expects the feedback owner to take action on a feedback. This default delay time is used to generate the Delay Analysis report.

To configure the default delay time:

  1. In the Drupal menu, point to Content Management, Content types, Edit feedback, Manage fields, and then click Delay time.
    The Delay Time page appears.

  2. Click to expand the Default value section.

  3. In the Delay Time box, enter the appropriate number of hours to set as the default delay time.

  4. Click Save field settings.
    The default delay time is updated in the OGPL Control Panel application. The updated default delay time appears when a VRM Administrator is assigning a feedback to a feedback owner.
