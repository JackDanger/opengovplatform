Viewing the List of Contrast Options

You can view the list of the contrast options that you created in the OGPL Control Panel application. The list displays the title of the contrast options, type of theme, and options to manage the contrast option.

Note: You cannot view the contrast options that are available by default, such as High Contrast and Standard Contrast.

To view the list of contrast options that you created:

  1. In the navigation menu, point to Configurable Items, Manage Header Content, and then click Manage Theme Files.
    The Theme Files page appears and displays the list of the contrast options that you created. The following table lists each field and its description.

Table: Description of fields in the Theme Files page

Field

Description

Title

Displays the title for the contrast options.

Themes

Displays the enabled theme to which you added this contrast option. For more information about enabled themes, see Enabling Themes.

Note: Currently, you can add a contrast option only to the ogpl_css3 theme.

Type

Displays the type of theme. For contrast options, this column displays Contrast.

Operations

Displays various tasks that you can perform to manage this contrast option, such as:

• View contrast option details.

• Edit contrast option.

• Delete contrast option.

 

Tip: Sorting the list helps you quickly analyze, visualize, and understand various contrast options. You can sort the list according to various criteria, such as the contrast option title, or type. To sort the list, click the appropriate column name.
