Adding Users

You can add users for the Open Government Platform Control Panel application and assign roles to these users by performing the following steps:

  1. In the Drupal menu, point to User Management, Users, and then click Add User.
    The Users page appears.

  2. Enter the appropriate information in each field. The following table lists each field and its description.
    Table: Description of fields in the Add User page

Field

Description

E-mail
(Required)

Enter the e-mail address that this user must use to log in to the OGPL Control Panel application.

Password 
(Required)

Enter a password for this user. Note: The OGPL Control Panel prompts the user to change this password at when the user the logs in for the first time.

Confirm password 
(Required)

Enter the same password that you entered in the Password box.

Status

Because you are adding a user, by default, this is set to Active.

If you want to block this user from logging in to the OGPL Control Panel application, click Blocked.

Roles

Select the check box of the role that you want to assign to this user.

Notes:

  • You can assign multiple roles to one user.

  • Every role has an associated default theme. When you assign a role to a user, the default theme is automatically assigned to the user.

Name 
(Required)

Enter the name of this user.

Phone Number 
(Required)

Enter the phone number of this user.

Agency 
(Required)

Click the government agency to which this user belongs.

 

  1. Click Create new account.
    This user can now access the OGPL Control Panel application using the e-mail address and password that you provide.
