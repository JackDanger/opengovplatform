Viewing the Workflow History for a Feedback

You can view and track the workflow history and status updates for each feedback. The History tab displays the workflow history, the date and time for each update, the username of the person who made the update, and the update performed in the workflow history, such as the feedback status changing from assigned to replied or a feedback forwarded to particular e-mail address.  

To track the workflow history for a feedback:

  1. In the navigation menu, click the Home link.
    The My List page appears.

  2. Click the tab of the feedback status for which you want to view the list of feedback. For example, if you want to view the list of closed feedback, click the Closed tab.

  3. In the row of the feedback for which you want to view the revision history, click the Preview this record icon .
    The Feedback Details page appears.

  4. Click the History tab.
    The History tab displays the workflow history for this feedback.
    Tip: If you want to sort the feedback history in ascending or descending order of the date and time for the updates, click the Time column name.
