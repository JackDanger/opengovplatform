Restricting Super Administrator Access

By default, Super Administrators can log on to the OGPL Control Panel Application from any computer. For security reasons, you can specify specific IP addresses from which Super Administrators can access the OGPL Control Panel Application.

To specify the IP addresses from which Super Administrators can access the application:

  1. In the Drupal menu, point to Site Configuration, and then click Site information.  
    The Site Information page appears.

  2. Scroll down until you see the Super Admin Access IPs box.

  3. In the Super Admin Access IPs box, enter the IP address for each computer from which you want to allow Super Administrators to access the OGPL Control Panel Application.
    Note: If you leave the Super Admin Access IPs box blank, Super Administrators can access the application from any computer.

  4. Click Save configuration.
    Super Administrators can now access the OGPL Control Panel Application through the IP addresses that you specified.

Note: You must ensure that you have added the IP address of the computer that you are using while specifying the IP addresses. If you do not add your IP address in the Super Admin Access IPs box and click Save configuration, a message appears on top of the page. You must enter your IP address and in the Super Admin Access IPs box and click Save configuration again.
