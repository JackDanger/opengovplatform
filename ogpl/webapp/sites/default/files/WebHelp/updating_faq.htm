Updating FAQs

An FAQ (Frequent Asked Question) consists of a question that is frequently raised by visitors to the OGPL Web site and the OGPL Web site administrator’s response to the question.

When a CMS Administrator assigns an FAQ to you, you must update it with the appropriate content and send it to the Moderator for review and approval.

The following image illustrates how an FAQ appears on the OGPL Web site.

To update the content for an FAQ:

  1. In the navigation menu, click the Home link.

  2. In the Content List section, click the Edit this record icon  for the FAQ that you want to update.
    The Edit page appears.

  3. Update the appropriate information in each field. The following table lists each field and its description.
    Table: Description of fields in the Edit page

Field

Description

Question

Update the question for this FAQ.

Answer

Update the answer for this FAQ.

Content Management

If you want to send the content to the Moderator, click Moderator Review.

If you want to retain the content with you for further update, click Create/Edit Content.

Comment

Enter additional information about this content. This information appears while viewing workflow history of this content.

Question details
(Optional)

Enter additional details for this question. This information appears with the answer of this question.

Revision Information - Log Message

Enter additional information about the updates that you made to the FAQ. This message appears while viewing the revision history for this content.

File Attachments

 Audio

Upload an audio file for this FAQ.

Note: You can upload MP3 files.

To upload an audio file:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the file that you want to upload.

  3. Click Open.
    The Audio box displays the path of the file that you selected.

  4. Click Upload.
    The file is uploaded to the database and a link to the audio appears on the page.

  5. Optional: If you want to remove this file, click Remove.
    The uploaded file is removed from the database.
Video

Upload a video file for this FAQ.

Note: You can upload MP4, MPEG, AVI, MPG, WMV, FLV, and MOV files.

To upload the video file:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the file that you want to upload.

  3. Click Open.
    The Video box displays the path of the file that you selected.

  4. Click Upload.
    The file is uploaded to the database and a link to the video appears on the page.

  5. Optional: If you want to remove the file, click Remove.
    The uploaded file is removed from the database.

Enter the Expiry Date

Update the date when the information in the FAQ page is outdated.

The content remains on the OGPL Web site until it is unpublished or deleted by the Super Administrator.

Notify Before (Days)

Update the number of days before the expiry date when you want to receive a notification about the expiry of the content.

    1. Click Save. The content record is updated.

Note: If you want to see how the content looks on the OGPL Web site before the content is published, click Preview.
