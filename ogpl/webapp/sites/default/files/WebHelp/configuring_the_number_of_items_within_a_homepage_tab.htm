Configuring the Number of Items Within a Homepage Tab

Homepage Tabs allow you to display tabbed content on the home page of the OGPL Web site. Each Homepage Tab retrieves and displays information from a content source. The following table lists each Homepage Tab and its content source.

Table: Description of Homepage Tabs and Content Source

Homepage Tab

Description and Content Source

Popular Catalogs

Displays most-viewed application catalogs, dataset catalogs, and document catalogs.

This Homepage Tab uses the Most_Popular_Catalog view as the content source. The Most_Popular_Catalog view in turn uses the following views to fetch the information:

  • Popular_Apps_Catalog

  • Popular_Dataset_Catalog

  • Popular_Document_Catalog

Recent Catalogs

Displays recently added application catalogs, dataset catalogs, and document catalogs.

This Homepage Tab uses the Most_Recent_Catalogs view as the content source. The Most_Recent_Catalog view in turn uses the following views to fetch the information:

  • Recent_Apps_Catalog

  • Recent_Dataset_Catalog

  • Recent_Document_Catalog

Major Events

Displays major events happening around the world through the RSS feeds.

This Homepage Tab uses a block as a content source. For example, RSS feed aggregator. For more information, see Managing RSS Feeds

 

You can configure the number of items that must appear on each Homepage Tab on the home page of the OGPL by performing the following steps:

  1. In the Drupal menu, point to Site building, Views, and then click List.
    The Views page appears and displays the list of views for different blocks of the OGPL Web site.

  2. In the views list, navigate to the content source for the Hompeage Tab that you want to configure, and click the Edit link. For example, if you want to edit the number of items that must appear on the Popular Catalogs Homepage Tab, navigate to the Popular_Apps_Catalog View, and click the Edit link.
    The Edit view View_Name page appears, where View_Name is the name of the View that you selected.

  3. In the Basic settings section, click the number link that appears next to Items to display.
    The Defaults: Items to display and Offset fields appear.

  4. In the Defaults: Items to display field, enter the number of items that must appear on the Homepage Tab. For example, if you want the names of the 10 most-viewed datasets to appear on the Popular Catalogs Homepage Tab, enter 10 in the Defaults: Items to display field.

  5. In the Offset field, enter the number of items that must not appear on the Homepage Tab. For example, if you enter 3 in the Offset field, the top 3 most-viewed datasets will not appear on the Popular Catalogs Homepage Tab.

  6. Click Update.
    The number of items that you specified appears in the appropriate Homepage Tab.
