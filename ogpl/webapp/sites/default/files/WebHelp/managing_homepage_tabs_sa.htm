Managing Homepage Tabs

The Rotating Tabs allow you to display the dynamic and static content in the form of Quicktabs. The Front End Home Page Rotating Tabs is a Drupal QT block and can consist of maximum 3 quicktabs.

The Front End Home Page Rotating Tabs block consists of 3 quicktabs:

  • Recent Catalogs: Displays the recently added catalogs.

  • Popular Catalogs: Displays the most viewed catalogs.

  • Major Events: Displays major events happening around the world through the RSS feeds.

The quicktabs use jQuery to fetch the information from the following content sources:

  • View: This content type is used to fetch and display the content for the Recent Catalogs and Popular Catalogs quicktabs.

  • Block: This content type is used to fetch and display the RSS feeds for the Major Events quicktab.

The quicktabs fetch information from the above content types and display on the home page of the OGPL Web site.
