Creating Country Data

You can add links of the OGPL Web site and the Open Data Site of another country in the OGPL Web site of your country. These links allow visitors to navigate to other government-related Web sites with minimal time and effort. The links are displayed as flags in the home page, and the URLs are listed in the WW Data Site page of the OGPL Web site. When a visitor clicks a flag or URL, the corresponding OGPL Web site or the World Wide Data Site page of that country appears.

Note: You must have the images of the flags for the country that you want to create.

The following image illustrates how Country Data appears on the OGPL Web site.

To add links and data of another country:

  1. In the navigation menu, click Configurable Items, and then click Create Content.
    The Create Content page appears.

  2. Click the Country Data link.
    The Create Country Data page appears.

  3. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Create Country Data page

Title

Description

Title
(Required)

Enter the name of the country for which you are creating this Country Data link. The name of the country appears as the ScreenTip for the flag image. This information also appears in the WW Datasite page.

Instructions

Enter the guidelines that the assigned Content Creator must follow while updating the content for this page. The Moderator and the Publisher reviews the content based on these guidelines.

Site Header Image

(Required)

Upload the image of the flag that must appear at the top left of the header of the OGPL Web site.

Note: The recommend image dimension is 38x24 pixels.

To upload an image:

  1. Click Browse.
    The File Upload page appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Site Header Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded to the database and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

Flag Image

Upload the image of the flag that must appear at the top right of the header of the OGPL Web site. The recommend image dimension is 22x23 pixels.

To upload an image:

  1. Click Browse.
    The File Upload page appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Flag Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded to the database and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

WW Data Site Image

(Required)

Upload the image of the flag that must appear at the top right of the header on the OGPL Web site. The recommend image dimension is 157x105 pixels.

To upload an image:

  1. Click Browse.
    The File Upload page appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Open Data Site Image box displays the path of the file that you selected.

  4. Click Upload.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.
