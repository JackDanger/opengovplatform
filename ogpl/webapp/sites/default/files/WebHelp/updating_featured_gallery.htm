Updating a Featured Gallery

The Featured Gallery displays important datasets on the OGPL Web site as a slide show. You can upload an image for the dataset and link the dataset page to the image.

The CMS Administrator uploads an image of a dataset to the Featured Gallery and assigns it to you. You link the image to the appropriate dataset, and send it to the Moderator for review.

The following image illustrates how Featured Gallery appears on the OGPL Web site.

To update the content for Featured Gallery:

  1. In the navigation menu, click the Home link.

  2. In the Content List section, click the Edit this record icon  for the Featured Gallery that you want to update.
    The Edit page appears.

  3. In this page, update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Edit page

Field

Description

Title
(Required)

Update title for this featured gallery image.

Content Management

If you want to send the content to the Moderator, click Moderator Review.

If you want to retain the content with you for further update, click Create/Edit Content.

Comment

Enter additional information about this content. This information appears while viewing workflow history of this content.

Revision information – Log message

Enter additional information about the updates that you made to this Featured Gallery. This message appears while viewing the revision history for this content.

Image

If you want to upload a new image for the dataset, do the following:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload to the dataset.

  3. Click Open.
    The Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. In the Alternate Text field, type a description of the image.
    This description is read by screen readers and search engines. If the image fails to load, this description appears on the OGPL Web site.

  6. In the Title field, type a title for the image.
    This text appears as screen tip for the image.

  7. Optional: If you want to remove this image, click Remove.
    The image is removed from the database.

High Value Datasets

Select the dataset that you want to link with the image that you uploaded.

Enter the Expiry Date

Update the date when the information in the Featured Gallery is outdated.

The content remains on the OGPL Web site until it is unpublished or deleted  by the Super Administrator.

Notify Before (Days)

Update the number of days before the expiry date when you want to receive a notification about the expiry of the content.

    1. Click Save. The content record is updated.

Note: If you want to see how the content looks on the OGPL Web site before the content is published, click Preview.
