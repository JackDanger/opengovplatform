Updating the Status for Various Feedback Simultaneously

Updating the status for each feedback separately involves significant time and effort. To avoid this, you can update the status of various feedback simultaneously with just a few clicks.

Note: You can only update the status of multiple feedback simultaneously for feedback that is in the Replied state or in the Reviewed state.  

To update the status for multiple feedback simultaneously:

  1. In the navigation menu, click the Home link.

  2. If you want to update the status of multiple feedback in the reviewed state, click the Reviewed tab.
    The Reviewed tab displays the list of reviewed feedback.
    If you want to update the status of multiple feedback in the replied state, click the Replied tab.
    The Replied tab displays the list of replied feedback.

  3. Optional: You can filter the feedback list. For more information, see Filtering the list of Feedback.

  4. Select the check box for each feedback for which you want to change the status.
    Tip: If you want to select all the feedback records currently displayed on the Web page, select the check box in the column header.

  5. If you want to archive all the selected feedback, in the Target Status list, click Archived.
    If you want to close all the selected feedback, in the Target Status list, click Closed.

  6. Click Change Status.
    The status of all the selected feedback is updated.
