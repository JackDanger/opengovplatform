Open Government Platform (OGPL) Control Panel Overview

The Open Government Platform (OGPL) Control Panel is a Web-based application that allows OGPL Web site Administrators to build OGPL Web sites, configure settings, add and manage users and their roles, configure the content management system, process the feedback from visitors of the OGPL Web site, and manage the content publishing workflow.

The OGPL Web site and the OGPL Control Panel application are built using Drupal, an open source content management platform. Drupal provides a wide range of inbuilt features that the OGPL Administrators can customize and leverage according to various requirements.

OGPL Web Site Overview

You can manage the OGPL Web site by using the OGPL Control Panel. The OGPL Web site is an Open Data Web site. Governments and their agencies can use the OGPL Web site to publish datasets, documents, services, tools, and applications for public use. The OGPL Web site enhances the transparency in the functioning of the Government and also opens avenues for innovative use of Government Data to build applications, conduct analyses, and perform research.

The OGPL Web site allows visitors to:

  • Easily search, view, download, and use datasets generated and held by the Government and their agencies.

  • View and navigate to the Open Government Web sites of other countries.

  • Provide feedback, comments, and recommendation about the content or functioning of the Web site to improve the Web site.

  • Rate and comment on current datasets.

  • Request for a dataset.
