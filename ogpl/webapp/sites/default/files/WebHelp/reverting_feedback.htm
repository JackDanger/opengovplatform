Reverting Feedback

You can revert a feedback to the Visitor Relationship Management (VRM) Administrator if you cannot perform the required action for the feedback.

To revert a feedback to the VRM Administrator:

  1. In the navigation menu, click the VRM link.
    The     VRM page appears.

  2. Optional: You can filter the feedback list. For more information, see Filtering the Feedback List.

  3. In the row of the feedback that you want to revert to the VRM Administrator, click the Edit this record icon .
    The Feedback Details Edit page appears.

  4. Click Move to “Reverted”.

The feedback is assigned to the VRM Administrator. The status of the feedback changes to Reverted.
