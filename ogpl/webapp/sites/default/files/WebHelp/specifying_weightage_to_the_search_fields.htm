Specifying the Weightage for the Search Fields

You can specify which fields are more important when searching for a keyword by giving those fields a greater weightage by entering a higher numeric value. The fields that have higher numeric value are given preference while listing the search results. For example, if you assign a greater numeric value to the text that appears within the H1 tag than the text that appears within H2 or H3 tag, the search results that have the matching keyword inside an H1 tag appear before the results that have the same keyword in H2 or H3 tag.

To specify the weightage for a search field:

  1. In the navigation menu, point to Configurable Items, Manage Header Content, and then click Manage Global/Advance Search.
    The Settings page appears.

  2. Click the Search Fields tab.
    The Apache Solr page appears and displays various content fields.

  3. In the Field biases section, click the appropriate numeric value that you want to assign as the weightage for each field that you want to give preference. For example, if you want the keyword that appears in the H1 tag to have more weightage than the same keyword in H2 or H3 tag, click 5 for Body text inside H1 tag and 1 for Body text inside H2 or H3 tags.

  4. Click Save Configuration.
    The preferences for the search fields are saved.
