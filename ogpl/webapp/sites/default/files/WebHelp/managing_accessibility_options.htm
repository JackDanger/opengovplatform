Managing Accessibility Options

Accessibility options help visitors who have visual disabilities to view text on the OGPL Web site. Using these options, a visitor can:

  • increase or decrease the size of the text on the OGPL Web page.
  • apply a color theme to the OGPL Web site.
  • view the content in standard contrast or high contrast.
