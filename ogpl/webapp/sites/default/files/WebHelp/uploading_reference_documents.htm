Uploading Reference Documents

You and the Program Management Office (PMO) can upload reference documents. These documents are only available on the OGPL Control Panel application and do not appear on the OGPL Web site.

To upload a reference document:

  1. In the navigation menu, point to Configurable Items, and then click Create Content.
    The Create Content page appears.

  2. Click the Reference Documents link.
    The Create Reference Documents page appears.

  3. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Create Reference Documents page

Title

Description

Title

(Required)

Enter the name of the document which you want to upload.

Description

(Required)

Enter a description about the document that you are uploading.

Input format

Click the format in which the Content Creator can update content in the Body field. For example: Filtered HTML, Full HTML, PHP Code, or Plain Text.

Document

Upload the document.

Note: You can upload TXT, DOC, DOCX, PDF, XLS, XLXS, RDF, XML, and PPT documents.

To upload the document:

  1. Click Browse.
    The File Upload page appears.

  2. Click the image that you want to upload to the dataset.

  3. Click Open.
    The Upload box displays the path of the file that you selected

  4. Click Upload.
    The file is uploaded to the database and a link to the file appears on the page.

  5. Optional: If you want to remove this document, click Remove.
    The uploaded document is removed from the database

Menu settings

You do not need to provide information for fields within this section.

URL path settings

By default, the Automatic alias check box is selected. The application automatically assigns the URL for this page.

If you want to assign a specific URL for this page, perform the following steps:

  1. Click to clear the Automatic alias check box.

  2. In the URL path settings box, type the URL that you want to assign for this page.

  1. Click Submit.

The reference documents are uploaded and are available to the users of the OGPL Control Panel application.
