Updating Country Data

The header section of the home page contains flags that link to the Open Data Site of another country. Country Data allow you to add information about each country, such as the flag image and the dataset link. When you create country data, the country information also appears in the WW Data Sites page, which displays all the Open Data Sites for every country. You can also upload the image of the flag that must be used as the Site Header image and appears on the top left of the OGPL Web site.

The following image illustrates how Country Data appears on the OGPL Web site.

You can update the information for a Country Data, before submitting the content to the Publisher, by performing the following steps:

  1. In the navigation menu, click the Home link.

  2. In the Content List section, click the Edit this record icon for the Country Data that you want to update.
    The Edit page appears.

  3. Update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Edit page

Field

Description

Title
(Required)

Update the name of the country for which you are creating this Country Data link. The name of the country appears as a ScreenTip for the image of the flag. This information also appears in the WW Datasite page.

Site Header Image
(Required)

Upload the image of the flag that must appear at the top left of the header of the OGPL Web site.

Note: The recommended image dimension is 38x24 pixels.

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Site Header Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

Flag Image

Upload the image of the flag that must appear at the top right of the header of the OGPL Web site.

The recommend image dimension is 22x23 pixels.

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Flag Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

Open Data Site Image
(Required)

Upload the image of the flag that must appear in the WW Data Sites page of the OGPL Web site.

The recommend image dimension is 157x105 pixels.

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload.

  3. Click Open.
    The Open Data Site Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

Union Government Name
(Required)

Update the Government name official name of the country that must appear on the OGPL Web site. This information also appears as ScreenTip on WW Datasite page.

Dataset Link

Update the URL of the dataset link of the country for which you are creating this Country Data content.

Latitude

Update the latitude coordinates of the country for which you are creating this Country Data content.

Longitude

Update the longitude coordinates of the country for which you are creating this Country Data content.

State/City/Agency Data

Region Type

Click the appropriate region such as State, City, Agency, Municipality and Sub-agency for the country from the list.

This information appears when a visitor clicks on the name of this country in the WW Datasite page.

State/Agency

Update the name of the state or agency of the selected region.

This information appears when a visitor clicks on the name of this country in the WW Datasite page.

This is a mandatory field if you add a City/sub-agency/Municipality.

Note: Once you update the State/Agency name and save this country data, the State/Agency name appears as autocomplete option when you update this country data and a new State/Agency.

City/Sub-agency

Update the name of the city, sub-agency, or municipality of the selected region.

This information appears when a visitor clicks on the name of this country in the WW Datasite page.

Note: This field is available only when you select City, Municipality or Sub-Agency as the Region Type and cannot be left blank.

Portal URL

Update the URL of the main portal link of the selected region.

Dataset URL

Update the URL of the dataset that belong to the selected region.

Latitude

Update the latitude coordinates for the selected region.

Longitude

Update the longitude coordinates for the selected region.

Content Management

If you want to approve the content, click Publisher Review.

If you want to send the content back to the Content Creator, click Create/Edit Content.

Comment

Enter additional information about this content. This information appears while viewing workflow history of this content.

Revision Information - Log Message

Enter additional information about the updates that you made to this Country Data. This message appears while viewing the revision history for this content.

Enter the Expiry Date

Update the date when the information of this Country Data is outdated.

The content remains on the OGPL Web site until it is unpublished or deleted by the Super Administrator.

Notify Before (Days)

Update the number of days before the expiry date when the Content Creator must receive a notification about the expiry of the content.

  1. Click Save.
    The content record is updated.

Note: If you want to see how the content looks on the OGPL Web site before the content is published, click Preview.
