Content Management System Moderator

CMS moderators can view the list of content that the CMS Administrator has assigned for review. If the content is appropriate, the CMS Moderator approves the content. If the content is inappropriate, the CMS Moderator must assign the content back to the Content Creator.

For more information on the CMS workflow, see CMS workflow.

The CMS Moderator can perform the following tasks:

  • View the list of content assigned for review and approval. For more information see, Viewing the Content List.

  • Filter the content list according to a specific content type and workflow step. For more information see, Filtering the List of Content.

  • Preview content. For more information, see Previewing the Content.

  • Approve content. For more information see, Approving Content.

  • Send content back to the Content Creator. For more information see, Sending the Content Back to the Content Creator.

  • View content revisions. For more information see, Viewing the Revision History of a Content.

  • Track content updates. For more information see, Tracking the Revisions to the Content.
