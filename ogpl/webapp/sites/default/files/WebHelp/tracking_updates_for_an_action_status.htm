Tracking Updates for an Action Status

You can view and track the updates made to an action status. The Action Status History page displays the details of the updates made to each action status.

To track the changes made to an action status:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Manage Actions tab.
    The Action Status List page appears.

  3. Click the History tab.
    The Action Status History page appears with the list of all the updates made to each action status. The following table lists each column and its description.

Table: Description of columns in the Action Status History page

Column

Description

Time

Displays the date and time when the VRM administrator modified the action status.

Details

Displays the details of the update, such as the user who performed this update and the update made to the action status.
