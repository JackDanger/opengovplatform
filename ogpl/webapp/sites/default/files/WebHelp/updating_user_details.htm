Updating User Details

You can update user information, such as e-mail address, password, status, user roles, default theme, or locale settings, by performing the following steps:

  1. In the navigation menu, point to Management, and then click User Management.
    The User Management page appears.

  2. In the row of the user whose details you want to edit, click the edit link under the Operations column.
    The Edit tab displays current details for the user.

  3. Update the appropriate user details. The following table lists each field and its description.

Table: Description of fields in the Edit tab

Field

Description

E-mail
(Required)

Update the e-mail address of this user.

All instances of this e-mail address in the OGPL Control Panel application are updated.

Password 
(Required)

If you want to assign a new password to this user, enter a password.

Confirm password
(Required)

Enter the same password that you entered in the Password box.

Status

By default, this is set to Active.

When the Status is set to active, the user can log in to and access the areas of the OGPL Control Panel application that are designated to the role that this user is assigned to.

If you want to block this user from logging in to the OGPL Control Panel application, click Blocked.

Roles

Select the appropriate check box to update the roles for the user.

Note: You can assign more than one role to a user.

Theme Configuration

Click the theme for the OGPL Web site that you want to display for this user.

Contact settings

If you want to allow this user to contact you using e-mail Select the Personal contact form option.

Note: Privileged users, such as OGPL Web site administrators can view your contact details even if you do not select this option.

Locale settings

Click the local time zone for this user. If you are not sure which time zone to select, choose the time zone that is closest to this user’s location that has the same rules for daylight saving time. This time zone is used to display the dates and times throughout the OGPL Web site.

 

  1. Click Save.
    The information for this user is updated in the OGPL Control Panel application.
