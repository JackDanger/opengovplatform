Configuring Global Settings for Themes

You can configure default display settings across the entire OGPL Web site, across all the themes. The settings that you specify in the Global Settings tab are used as the default settings, unless you override them using a specific theme, such as a CMS theme.

Note: While configuring a theme, if you click Reset to defaults, all the theme settings are reset to the values that are configured in the Global Settings tab.

To configure the global settings for themes:

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Themes.
    The Themes page appears with a list of the all the themes currently available in the OGPL Control Panel.

  2. In the row of the theme that you want to configure, click the configure link.
    The Configure tab displays the settings for the theme that you selected.

  3. Click the Global Settings tab.
    The Global Settings tab displays global settings across all the themes in the OGPL Control Panel.

  4. Configure the appropriate global settings. The following table lists each field and its description.

Table: Description of fields in the Global Settings tab on Themes page

Field

Description

Toggle Display

Select the check boxes for page elements that you want to display on the OGPL Web site, and clear the check boxes for page elements that you do not want to display.

For example, if you want to display a logo on the OGPL Web site, select the Logo check box.

Display post information on

Select the check boxes for the types of posts for which you want to display the text: submitted by <Username> on <date>. This text displays the user name of the author of the article and the date on which the article was submitted.

Logo image settings

This section enables you to specify a logo for OGPL Web site.

Use the default logo

If you want to use the default logo on the OGPL site, select this check box.

Path to custom logo

If you want to another logo instead of the default logo, enter the path to the image that you want to use.

Upload logo image

Upload the image that you want to use as default logo for the OGPL Web site.

To upload an image:

  1. Click Browse. The File Upload dialog box appears.

  2. Browse for and select the image that you want to upload.

  3. Click Open.

  4. Click Upload. The image is uploaded and a preview of the image is displayed.

Shortcut icon settings

This section enables you to specify a shortcut icon to display for OGPL Web site URL.

Use the default shortcut icon

If you want to use the default shortcut icon for the OGPL Web site URL, select this check box.

Path to custom icon

Enter the path to the image you want to use as your shortcut icon file instead of the default shortcut icon.

Upload icon image

If you do not have direct file access to the OGPL Web server, you can upload an image that you want to use as shortcut icon for the OGPL Web site URL.

To upload an image:

  1. Click Browse. The File Upload dialog box appears.

  2. Browse for and select the image that you want to upload.

  3. Click Open.

  4. Click Upload. The image is uploaded and a preview of the image is displayed.

 

  1. Click Save configuration.
    The global settings for themes are saved in the OGPL Control Panel application.
