Forwarding a Feedback

You can forward a feedback to PMOs, POCs, and to people who are external to the OGPL Control Panel application.

Note: You can only forward a feedback that is assigned to you.

To forward a feedback:

  1. In the navigation menu, click the VRM link.
    The VRM page appears.

  2. Click the Edit this record icon for the feedback that you want to forward.
    The Feedback Details Edit page appears.

  3. Click Forward.
    The Forward Feedback page appears.

  4. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Forward Feedback page

Title

Description

Forwarded By

(Required)

Displays your e-mail address.

Note: You cannot update this field.

Send to

(Required)

Displays the e-mail address of each person to whom you are forwarding this feedback.

PMO

If you want to forward this feedback to a PMO, click the username of the appropriate PMO.

Note: To add more than one recipient, hold down CTRL and click each username.

The e-mail address corresponding to the username that you selected appears in the Send to box

POC

If you want to forward this feedback to a POC, click the username of the appropriate POC.

Note: To add more than one recipient, hold down CTRL and click each username.

The e-mail address corresponding to the username that you selected appears in the Send to box.

Add new emails

Type the e-mail address of any other person to whom you want to forward this feedback.

Note: Separate each e-mail address using a comma.

Subject

(Required)

Type an appropriate subject for the feedback e-mail.

Comments

(Required)

Type additional comments that you want to include in the feedback e-mail

  1. Click Submit.

The feedback is forwarded to the selected recipients.
