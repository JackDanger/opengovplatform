Viewing the List of E-mail Templates

The Previewable Email Templates page displays available e-mail templates for forwarding a feedback, replying a feedback and expiry notification with their details, such as the name, title, and recipient callback.

To view the list of e-mail templates, in the Drupal menu, point to Site building, and then click Previewable email templates.

The Previewable Email Templates page appears. The List tab displays the list of e-mail templates with their details. The following table lists each column and its description.

Table: Description of columns in the Previewable email templates page

Columns

Description

Name

Displays the name for this e-mail template. Drupal uses this name in URLs and API calls.

Title

Displays the descriptive title for this e-mail template.

Recipient callback

Displays the name of the function that is called to retrieve the list of recipients, if used in this template.

Custom Tokens

Displays the custom tokens that you created for this e-mail template using Drupal module.

Operations

Displays options to edit or delete this e-mail template.
