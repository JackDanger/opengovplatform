Adding Color Themes

You can add a color theme for a theme that you enabled on the Manage Themes page of the OGPL Control Panel application. For more information about enabled themes, see Enabling Themes.

Note: Currently, you can add a color theme only to the ogpl_css3 theme.

To add a new color theme:

  1. In the navigation menu, point to Configurable Items, Manage Header Content, and then click Manage Theme Files.
    The Theme Files page appears.

  2. Click Add.  
    The Create Theme Files page appears.

  3. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Create Theme Files page

Field

Description

Title

(Required)

Enter an appropriate title for this color theme. This title appears on the Theme Files page of OGPL Control Panel application.

Themes

(Required)

Click the enabled theme to which you want to add this color theme. For more information about enabled themes, see Enabling Themes.

Note: Currently, you can add a color theme only to the ogpl_css3 theme.

Type
(Required)

Click Color.

Theme File
(Required)

Upload the color theme file. The theme file must be a CSS file and the file size must not exceed 5 MB.

To upload a theme file:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the theme file that you want to upload.

  3. Click Open.
    The Theme File box displays the path of the file that you selected.

  4. Click Upload.
    The theme file is uploaded.

Image

Upload the image file required for this color theme file, such as the background image for this theme. The image file must be a PNG, GIF, or JPEG file and the file size must not exceed 5 MB.

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image file that you want to upload.

  3. Click Open.
    The Image box displays the path of the file that you selected.

  4. Click Upload.
    The image file is uploaded.

Note: If you want to upload another image for this theme, click Add another item.

Optional: If you want to remove this image, click Remove.
The uploaded image is removed from the database.

Logo

(Required)

Upload the image file that you want to use as the color theme icon for this theme on the OGPL Web site. The image file must be a PNG, GIF, or JPEG file and the file size must not exceed 5 MB.

To upload a color theme icon image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image file that you want to upload.

  3. Click Open.
    The Image box displays the path of the file that you selected.

  4. Click Upload.
    The image file is uploaded.

Optional: If you want to remove this image, click Remove.
The uploaded image is removed from the database.

 

  1. Click Submit.
    The color theme is saved and its color theme icon appears on the OGPL Web site. Visitors to the OGPL Web site can click the color theme icon to apply this color theme.
