Viewing the Top Search Phrases Report

When users are not sure where content is located on a Web site, they use the search feature to locate the content. If a search word or phrase in ranked high in the Top Search Phrase report and the content is not available on the OGPL site, you can create the content. If the content for the search word already exists on the OGPL Web site, the visitors are unable to find it. In this case, you might need to update the location of the content.

The Top Search Phrases report lists all the words or phrases that the visitors searched for on the OGPL Web site.  

To view the Top Search Phrases report, in the navigation menu, point to Reports, CMS Reports, Access Statistics, and then click Top Search Phrases.

By default, the words or phrases that were searched for the most number of times appears at the top of the Top Search Phrases report. You can sort the records by clicking the appropriate column name.
