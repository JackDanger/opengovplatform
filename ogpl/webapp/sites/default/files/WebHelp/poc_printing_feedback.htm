Printing Feedback Details

You can print a feedback details page. Depending on the feedback source, feedback category, and other feedback details, the print version can include the following details for a feedback:

  • Name of the visitor who sent the feedback. The visitor name will not display if the visitor sent this feedback from the Ratings page on the OGPL Web site.

  • E-mail address of the visitor who sent this feedback. The e-mail address will not display if the visitor who sent this feedback decided not to provide the e-mail address.

  • Subject, if the feedback was sent from the Contact Us page.

  • Feedback category. The feedback category will not display if no category has been allocated to this feedback.

  • Feedback text

  • User name to whom the feedback is assigned. The user name will not display if a feedback owner is not selected for this feedback.

  • Source from where the feedback originated, such as Feedback page or Contact Us page

  • Delay time, in hours

  • Current status of the feedback

  • Feedback Action status, if the VRM Administrator, PMO or POC  has selected an Action status.

  • Dataset, if the feedback originated from Ratings page on the OGPL Web site

  • Reply or review comments from the POC, PMO, or VRM

  •  Notes added by the VRM Admin, Assignee and users to whom the feedback was forwarded.

To print the details for a feedback:

  1. In the navigation menu, click the Home link.
    The My List page appears.

  2. To view the list of all feedback, click the All tab. The page that displays the list of all feedback appears.
    –or–
    Click the tab of the feedback status for which you want to view the list of feedback. For example, if you want to view the list of unassigned feedback, click the Opened tab.

  3. In the row of the feedback that you want to print, click the Preview this record icon .
    The Feedback Details page appears.

  4. Click the Print icon .
    A print preview of the feedback appears.
    The Print dialog box appears and displays the print options. You can update any of these options according to your print requirements.

  5. Click Print.
    The feedback details are printed.
