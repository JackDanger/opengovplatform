Configuring Image Captcha

Image CAPTCHA is a CAPTCHA type where random text is mixed with an image. You can configure various Image CAPTCHA settings, such as code settings, font settings, color and image settings, and distortion and noise settings.

Note: The image generation per CAPTCHA is a CPU-intensive activity. It is recommended to use moderate size and computation-related settings for the Image CAPTCHA.

To configure the Image CAPTCHA:

  1. In the navigation menu, point to Configurable Items, and click Manage Captcha.
    The CAPTCHA page appears.

  2. Click the Image CAPTCHA tab.
    The Image CAPTCHA tab displays the settings required to configure this CAPTCHA.

  3. Configure the appropriate Image CAPTCHA settings. The following table lists each field and its description.

Table: Description of fields in the Image CAPTCHA tab on CAPTCHA page

Field

Description

Example

Displays an example of an Image CAPTCHA.

Code settings

Use this section to select the characters that you want to use in the Image CAPTCHA and set the length of the CAPTCHA.

Characters to use in the code

Enter the characters that you want to include in generation of Image CAPTCHA. For example, you can include all the alphabets (A-Z, a-z) and numbers (0-9).

Code Length

Click the number to specify the length of the CAPTCHA. The minimum length required is 2 and the maximum length allowed is 10. The code length affects the size of the image. Note that larger values make the image generation more CPU-intensive.

Font Settings

Use this section to configure the font, font size, font color and other related font settings required for the Image CAPTCHA.

Fonts

Select the check box for each font that you want to use for the text in the Image CAPTCHA.

Note: If you want to use your own TrueType fonts, add them to the following OGPL Web site directory:

sites/all/libraries/fonts or sites/default/libraries/fonts

Font size

Click the number for specify length of the CAPTCHA text. Minimum font size required is 9 points, and maximum font size allowed is 64 points. The font size determines the size of the image, and larger values make the CAPTCHA generation more CPU-intensive.

Character Spacing

Click the option to specify the average spacing between characters in the CAPTCHA.

For example, if you want the characters in the CAPTCHA to be very closely spaced, click tight.

Color and image settings

Use this section to configure the background, text colors, and the file format of the image CAPTCHA.

Background color

Enter the hexadecimal code for the background color of this CAPTCHA. For example, if you want to use a white background for the CAPTCHA, enter #FFF or #FFFFFF.

Text Color

Enter the hexadecimal code for the color of the text in this CAPTCHA. For example, if you want to display the CAPTCHA text in black color, enter #000 or #000000.

Additional variation of text color

Click the option to specify the extent to which the characters have random colors in the range that you specified for the text color.

For example, if you want each character in the CAPTCHA to have a random color around the color that you specified in the Text Color, click High.

File format

Click the file format that you want to use for the images in this CAPTCHA.

Distortion and noise

Use this section to configure distortion and noise settings for Image CAPTCHA.

Distortion level

Click the option to specify the degree of wave distortion in the image. For example, if you want the image to be very highly curved and distorted, click 10-high.

Add salt and pepper noise

If you want to add randomly colored point noise, select this check box.

Add line noise

If you want to add lines randomly drawn on top of the text code, select this check box.

 

  1. Click Save configuration.
    The settings for Image Captcha are saved in the OGPL Control Panel application.
