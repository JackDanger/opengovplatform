Republishing Content

You can republish content to the OGPL Web site. After you republish content, the content appears in the pre-assigned area on the OGPL Web site and the URL of the content page is activated.

Note: If the area that is specified for the republished content is allocated to another content, the republished content does not appear on the OGPL Web site.

To republish content:

  1. In the Drupal menu, point to Content Management, Content, and then click List.
    The Content page appears.

  2. To search for the content that you want to republish, click the appropriate filter options.

  3. Under the Title column, select the check box for each content record that you want to republish on the OGPL Web site.

  4. In the Update options list, click Publish.

  5. Click Update.
    The content is republished to the OGPL Web site.
