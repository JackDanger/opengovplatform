Updating the Delay Time

You can update the delay time for the new feedback coming into the system by performing the following steps:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Manage Delay Time tab.
    The Manage Delay Time page appears.

  3. In the Default Delay Time (Required) box, update the number of hours that you want to set as the default delay time.

  4. Click Update.
    All the new feedback that you receive after the update have the updated delay time.
