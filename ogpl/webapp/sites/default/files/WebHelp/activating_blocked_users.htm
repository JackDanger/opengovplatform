Activating Blocked Users

You can activate a user that you blocked from accessing the OGPL Control Panel application. After you activate a blocked user, the user can perform all the tasks associated with the role that is assigned to the user.

To activate a blocked user:

  1. In the Drupal menu, point to User Management, Users, and then click List.
    The Users page appears.

  2. In the Users page, under the Operations column, click the Edit link for the user name that you want to activate.
    A page that displays the user account information appears.

  3. Scroll down until you see the Status box, and then click Active.

  4. Click Save.
    The user is activated and can access the OGPL Control Panel application.
