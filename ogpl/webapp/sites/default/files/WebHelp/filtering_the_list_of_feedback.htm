Filtering the List of Feedback

To view specific feedback from users of the OGPL Web site, you can specify various filter criteria, such as the feedback source, category, date, or source.

To filter the list of feedback:

  1. In the navigation menu, click the Home link.

  2. In the VRM page, click the All tab.
    The All Feedbacks page appears.
     – or –
    Click the tab of the feedback status for which you want to view the list of feedback. For example, if you want to view the unassigned feedback, click the     Opened tab.

  3. Click the Filters link.
    The Filters section expands. The following table lists each field and its description.

Table: Description of fields in the Filters section

Tab

Description

Category

Click the feedback category for which you want to view the list of feedback received.

For example, if you want to view the list of feedback related to applications, tools, and services, click App/Tools/Services.

From date

Click the start date of the duration within which you want to view the list of feedback received.

By default, this field displays the date 30 days prior to the current date.

For example, if you want to view the list of feedback received between January 1, 2012 and January 31, 2012, click January 1, 2012.

To date

Click the end date of the duration within which you want to view the list of feedback received.

By default, this field displays the current date as the To date.

For example, if you want to view the list of feedback received between January 1, 2012 and January 31, 2012, click January 31, 2012.

Source

Click the source for which you want to view the list of feedback received.

For example, if you want to view the list of feedback received from the Contact Us page, click Contact Us.

Status

Click the feedback status for which you want to view the list of feedback received.

For example, if you want to view the list of feedback assigned to an owner, click Assigned.

Note: This filter criteria is only available in the My List tab.

  1. Click Apply.
    The feedback matching the filter criteria appears on the feedback list.
