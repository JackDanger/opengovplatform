Viewing Menu Items in a Menu

You can view the menu items configured under a menu by performing the following steps:

  1. In the navigation menu, point to Configurable Items, and then click Manage Menu.
    The Menus page appears.

  2. In the List menus tab, click the menu for which you want to view the menu items.
    The List items tab displays the menu items available in the menu that you selected. The following table lists each field and its description.
    Table: Description of fields in the List Menus tab

Field

Description

Menu item

Displays the name of the menu item. When you click the menu item name, the corresponding OGPL Web page appears.

Enabled

Indicates whether this menu item is currently visible on the OGPL Web site.
If you want the menu item to appear on the OGPL Web site, select this check box.

If you do not want the menu item to appear on the OGPL Web site, clear this check box.

Expanded

Indicates whether this menu item has its sub-menus listed on the OGPL Web site.
If you want this menu item with its sub-menus expanded on the OGPL Web site, select this check box.

If you do not want the menu item with its sub-menus to appear expanded on the OGPL Web site, clear this check box.
