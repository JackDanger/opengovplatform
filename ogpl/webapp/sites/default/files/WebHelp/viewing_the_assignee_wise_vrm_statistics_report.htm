Viewing the Assignee Wise VRM Statistics Report

The Assignee Wise VRM Statistics report lists the total number of feedback for each assignee. This report enables you to track the number of feedback that each assignee has replied to, reviewed, closed, and archived.

You can also view the Assignee Wise VRM Statistics report for a specific duration by performing the following steps:

  1. In the navigation menu, point to Reports, VRM Agency Reports, and then click Assignee Wise VRM Statistics.
    The Assignee Wise VRM Statistics report for all the feedback received till date appears.

  2. In the From date box, click the start date of the duration within which you want to view this report.
    For example, if you want to view the VRM Delay Analysis report between January 1, 2012 and January 31, 2012, click January 1, 2012.

  3. In the To date box, click the end date of the duration within which you want to view this report.
    For example, if you want to view the VRM Delay Analysis report between January 1, 2012 and January 31, 2012, click January 31, 2012.

  4. Click Submit.
    The Assignee Wise VRM Statistics report for  the selected date range appears.
