Editing Feedback Status

If you want to provide more information about a feedback status, you can update the description of the feedback status.

To update the description of a feedback status:

  1. In the navigation menu, click the Home link.
    The VRM page appears.

  2. Click the Status tab.
    The VRM Workflow Status page appears and displays the list of workflow status.

  3. In the Operations column, click the edit link for the feedback status that you want to update.
    The Edit Status Description page appears.

  4. In the Description field, update the description for the feedback status.

  5. Click Save.
    The feedback status description is updated and appears in the VRM Workflow Status page.
