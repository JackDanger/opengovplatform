Updating Menus

You can update a menu by changing the menu name or the menu description.

To update a menu:

  1. In the navigation menu, point to Configurable Items, and then click Manage Menu.
    The Menus page appears.

  2. In the List menus tab, click the menu that you want to update.
    The List items tab displays the menu items available in the menu that you selected.

  3. Click the Edit menu tab.
    The Edit menu tab displays the current menu title and description.

  4. In the Title box, update the name for the menu.

  5. Optional: In the Description box, update the description for the menu.

  6. Click Save.
    The menu is updated as per the details that you specify. You can view the updated entry in the List menus tab.
