Configuring Mini Panels

You can configure default settings for Mini Panels and specify the Mini Panels for displaying new content that appears on the OGPL Web site. You can also select the layout options to be made available to the OGPL Control Panel users to choose from when selecting a layout for their content.

To configure the settings:

  1. In the Drupal menu, point to Site building, Mini panels, and then click Add.
    The Mini Panels page appears.

  2. Select the appropriate options. The following table lists each field and its description.

Table: Description of fields in the Mini Panels page

Field

Description

New content behaviour

Select the check boxes for the Mini Panels on which you want to display new content.

Select allowed layouts

Select the check boxes for the layouts that you want to allow OGPL Control Panel users to choose from when selecting a layout for their content.

Note: You must select at least one check box.

 

  1. Optional: If you only want to temporarily update the latest updates without saving them to the database, click Update.
    Note: The updates for this Mini Panel are not saved to the database until you click Save.

  2. Click Save.
    The Mini Panel settings are saved in the OGPL Control Panel application.
