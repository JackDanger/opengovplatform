Configuring Site Information

You can configure OGPL Web site settings, such as the site name, e-mail address for automated e-mails, and home page URL, by performing the following steps:

  1. In the navigation menu, point to Management, Layout Management, and then click Manage Site Information.
    The Site Information page appears.

  2. Enter or select the appropriate site settings. The following table lists each field and its description.

Table: Description of fields in the Site Information page

Field

Description

Name
(Required)

Enter a name for this OGPL Web site.

E-mail address (Required)

Enter the e-mail address where automated e-mails for registration, password requests, and notifications must be sent.

Note: To avoid flagging these e-mails as spam, use an e-mail address based on the site domain.

Mission

Enter the goals and objectives of this OGPL Web site.

Footer message

Enter the text that you want to display in the footer section of all the OGPL Web pages.

Tip: This is a good place to provide copyright information.

Anonymous user (Required)

Enter a title for users who are not logged in to the site. For example, Anonymous User or Guest User.

Default front page
(Required)

Enter the URL of the page that you want to display as the home page of the OGPL Web site.  

Select the Country

Click the country name for which you want to display the country flag and the government name on the OGPL Web site.

Automatically run cron

Click the time interval when you want to run the scheduled tasks on the OGPL database. For example, if you want to run the scheduled database tasks every hour, click 1 hour.

 

3. Click Save configuration.
The site information settings are saved in the OGPL Control Panel application.
