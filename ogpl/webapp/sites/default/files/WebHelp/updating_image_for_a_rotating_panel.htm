Updating Images for a Rotating Panel

The Rotating Panel on the home page of the OGPL Web site is a series of images of various government-related Web sites or government-initiated programs and schemes. When a visitor clicks an image on the rotating panel, the corresponding Web site appears. A Content Creator can upload an image for a Web site to the Rotating Panel, and specify the URL for that Web site.

The following image illustrates how a Rotating Panel appears on the OGPL Web site.

You can update the images or the link to the Web site before submitting the content to the Publisher, by performing the following steps:

  1. In the navigation menu, click the Home link.

  2. In the Content List section, click the Edit this record icon for the Rotating Panel that you want to update.
    The Edit page appears.

  3. In this page, update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Edit page

Field

Description

Title
(Required)

Update the title for this Rotating Panel image.

Description

Update the description about the image and the Web site it must link to.

Image

If you want to update the image of the Web site for the full view of the rotating panel, do the following:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload to the dataset.

  3. Click Open.
    The Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The image is removed from the database.

Smaller_image

Upload the image for the Web site that must be used for the half view of the rotating panel.

URL

Enter the new URL of the Web site that the image must link to.

Body

Update the textual content that the Content Creator submitted for your approval.

Content Management

If you want to approve the content, click Publisher Review.

If you want to send the content back to the Content Creator, click Create/Edit Content.

Comment

Enter additional information about this content. This information appears while viewing workflow history of this content.

Revision Information - Log Message

Enter additional information about the updates that you made to this Rotating Panel. This message appears while viewing the revision history for this content.

Enter the Expiry Date

Update the date when the information in this Rotating Panel is outdated.

The content remains on the OGPL Web site until it is unpublished or deleted b by the Super Administrator.

Notify Before (Days)

Update the number of days before the expiry date when the Content Creator must receive a notification about the expiry of the content.

  1. Click Save.
    The content record is updated.

Note: If you want to see how the content looks on the OGPL Web site before the content is published, click Preview.
