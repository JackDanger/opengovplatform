Point of Contact

The Point of Contact (POC) is a representative of an agency that works with a government office. The POC can perform the following tasks for each feedback:

  • View the list of feedback that the VRM Administrator assigned. For more information, see <Viewing the List of Feedback>

  • Filter the Feedback List. For more information, see Filtering Feedback List.

  • Assign a feedback action to a feedback. For more information, see <Assigning Feedback Category>

  • Update the feedback action of a feedback. For more information, see <Updating Feedback Category>

  • Reply to a feedback. For more information, see <Replying to Feedback>

  • Review feedback. For more information, see <Reviewing a feedback>.

  • Forward a feedback to another Program Management Office or Point of Contact. For more information, see <Forwarding  Feedback>

  • Add notes to a feedback. For more information, see Adding Notes to Feedback.

  • Revert a feedback to the VRM Administrator. For more information, see Reverting Feedback.

  • View revision history of a feedback. For more information, see <Viewing Revision History>.

 

Note: The POC cannot perform any administrative tasks.
