Deleting Users

When you delete a user, all instances of the user in the CMS module appear as Anonymous. After deleting a user, you must reassign the work items that were assigned to this user.

To delete a user account:

  1. In the navigation menu, point to Management, and then click User Management.
    The User Management page appears.

  2. In the row of the user account that you want to delete, click the delete link.
    A confirmation message appears.

  3. Click Delete.
    The user is deleted from the OGPL Control Panel application.
