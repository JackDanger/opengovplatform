CMS Reports

CMS reports are tabular and graphical representations of various content-related metrics, such as Error logs and Access Statistics. These reports also enable you track the errors that the visitors encountered while visiting the OGPL Web site and traffic statistics for the OGPL Web site.

You can view the following reports for the CMS workflow:

  • Access Denied Pages/Datasets Report
  • Page Not Found Error Report
  • Top Pages Report
  • Top Referrer Report
  • Top Visitors Report
  • Top Search Phrases Report
