Assigning a Feedback

You can assign a feedback to the Program Management Office (PMO), Point of Contact (POC), or Visitor Relationship Management (VRM) Administrator. When assigning the feedback, you must consider the content of the feedback and the action that the feedback owner must perform to address the feedback.

If the feedback owner is unable to address the feedback, the owner returns the feedback to you. You can then address the feedback or reassign the feedback to a different owner.

To assign a feedback:

  1. In the navigation menu, click the Home link.

  2. Click the All tab.
    A page that displays the list of all the feedback appears.
    – or –
    To view the list of unassigned feedback, click the Opened tab.

  3. Optional: You can filter the feedback list. For more information, see Filtering the List of Feedback.

  4. In the row of the feedback that you want to assign, click the Edit this record icon.
    The Feedback Details Edit page appears.

  5. View or update the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Feedback Details Edits page

Displays the subject of this feedback. If the visitor has not provided a subject while submitting the feedback, this field appears blank.
