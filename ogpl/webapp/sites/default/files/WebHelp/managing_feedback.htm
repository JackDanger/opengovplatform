Managing Feedback

Users can provide feedback through various pages of the OGPL Web site, such as the Contact Us page, Feedback page, Ratings page, and Suggest Dataset page. When you log on to the application, the list of feedback received automatically appears on the page. You must take the appropriate action for each feedback. For more information on the feedback workflow, see Feedback Workflow.

You can perform the following tasks for each feedback:

  • View the list of feedback. For more information, see Viewing Feedback.

  • Assign a feedback to the Program Management Office, Point of Contact, or VRM Administrator. For more information, see Assigning feedback.

  • Reassign feedback that the Program Management Office or Point of Contact reverted. For more information, see Reassigning the feedback.

  • Forward a feedback to the Program Management Office or Point of Contact. For more information, see Forwarding the Feedback.

  • Assign a feedback category to a feedback. For more information, see Assigning a Feedback Category.

  • Add, edit, delete, and track a feedback action status. For more information, see Managing the Action Status for a Feedback.

  • Reply to a feedback. For more information, see Replying to the feedback.

  • Mark a feedback as Irrelevant. For more information, see Marking a feedback as Irrelevant.

 

You can also perform the following administrative tasks:

  • Add, edit, delete, and track a feedback category. For more information, see Managing Feedback Categories.

  • Set, modify, and track the delay time for responding to a feedback. For more information, see Managing Feedback Delay Time.

  • Update and track the status for a feedback. For more information, see Managing Feedback Status.
