Resetting Text Resize Settings

Resetting text size settings to default values removes all the latest changes that you made in the Text Size Configuration Web page and restores all default values.

To reset text size settings to their default values:

  1. In the navigation menu, point to Configurable Items, Manage Header Content, and then click Manage Accessibility (Text Resize).
    The Text Size Configuration page appears.

  2. Scroll down until you reach the end of the page, and then click Reset to defaults.
    The text size settings are reset to their default values.
