Viewing the List of RSS Feeds

You can view a list of RSS feeds currently available in the OGPL Web site and other details, such as the time when a RSS feed was last updated and time remaining until the next update of the RSS feed.

To view the list of RSS feeds:

  1. In the navigation menu, point to Configurable Items, and then click Manage RSS Aggregator.
    The Feed Aggregator page appears. The List tab displays a list of all the RSS feeds configured in the OGPL Web site. The following table lists each field and its description.

Table: Description of fields in the Settings tab

Field

Description

Title

Displays the title for the RSS feed, as it appears on the OGPL Web site.

Items

Displays the number of feed items available in this RSS feed.

Last update

Displays the number of days since this RSS feed was last updated.

Next update

Displays the time remaining for the next update of this RSS feed.

Operations

Displays the options for the tasks you can perform for this RSS feed, such as updating this RSS feed, updating the items in this RSS feed, and removing items from this RSS feed.

For more information about updating this RSS feed, see Updating RSS Feeds.

For more information about updating items in this RSS feed, see Updating Items in RSS Feeds.

For more information about removing items from this RSS feed, see Removing Items from RSS Feeds.
