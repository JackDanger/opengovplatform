Filtering the User List

To view specific users, you can filter the users list according to a user role or user account status.

To filter the User List:

  1. In the navigation menu, point to Management and then click User Management.
    The User Management page appears.

  2. If you want to view the list of users that belong to a particular user role, in the Role list, click the appropriate user role.
    If you want to view the list of users according to their account status, in the Active list, click the appropriate option.
    For example, if you want to view the list of active Moderators, in the Role list, click Moderator, and then in the Active list, click Yes.

  3. Click Apply.
    The User Management page displays the list of users that matches the filter criteria that you specified.
