Configuring reCaptcha

You can use reCAPTCHA Web service to improve the CAPTCHA system and protect e-mail addresses. To use reCAPTCHA, you need to first register at www.recaptcha.net. After you register, the page displays the public key and private key that you can use for the reCAPTCHA settings in the OGPL Web site.

You can configure various reCAPTCHA settings, such as the public key and private key, secure connection settings, and theme settings.

To configure the reCAPTCHA:

  1. In the navigation menu, point to Configurable Items, and click Manage Captcha.
    The CAPTCHA page appears.

  2. Click the reCAPTCHA tab.
    The reCAPTCHA tab displays the settings required to configure this CAPTCHA.

  3. Configure the appropriate reCAPTCHA settings. The following table lists each field and its description.

Table: Description of fields in the reCAPTCHA tab on CAPTCHA page

Field

Description

Public Key

(Required)

Enter the public key that was provided when you registered at reCAPTCHA.net.

Private Key
(Required)

Enter the private key that was provided when you registered at reCAPTCHA.net.

Secure Connection

If you want to connect to the reCAPTCHA server using a secure connection, select this check box.

AJAX API

If you want to use the AJAX API to display reCAPTCHA, select this check box.

Theme Settings

Use this section to configure the theme and tab index settings for reCAPTCHA.

Theme
(Required)

Click the visual theme that you want to use for reCAPTCHA. 

Tab Index

If you want the reCAPTCHA box to be selected in a particular tab sequence, enter the appropriate tab index. Tabindex is a number that specifies the order in which various page elements are selected when a user presses the TAB key on the keyboard. By default, this box is blank and the default tab sequence is followed.

 

  1. Click Save configuration.
    The settings for reCAPTCHA are saved in the OGPL Control Panel application.
