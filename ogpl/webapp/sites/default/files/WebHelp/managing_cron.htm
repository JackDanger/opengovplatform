Managing the Cron Job

Cron is a time-based scheduler that enables you to schedule jobs to run periodically at a certain time or date. A cron is configured in the OGPL Control Panel application to perform the following tasks:

  • Instruct Apache Solr to index new and updated content, and store the keywords from the content in an index file. New content does not appear on the OGPL Web site until Apache Solr indexes the content.
  • Search for the content that reached its expiry notification date and send an e-mail notification to the assigned Content Creator. This e-mail is sent only once.

Note: When the cron job is executed, both these tasks are carried out simultaneously.
