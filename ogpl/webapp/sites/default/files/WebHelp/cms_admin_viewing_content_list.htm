Viewing Content List

When you log in to the OGPL Web site, the Content List section displays the list of content that you created and the content details, such as the content type, the user name of the assigned content creator who updates the content, and the user name of the moderator who reviews the content. 

To view the content list:

  1. In the navigation menu, click the Home link.

  2. Browse through the Content List section. The following table lists each field and its description.

Table: Description of fields in the Content List page

Field

Description

Title

Displays the name of the content that you created.

Type

Displays the content type, such as Page, Block, FAQ, Featured Gallery Rotating Panel or Country Data.

Updated date

Displays the date when the content was last updated.

Content Creator

Displays the username of the Content Creator that you assigned to update this content.

Moderator

Displays the username of the Moderator that you assigned to review this content.

Duration

Displays the number of days since when you assigned the content to the Content Creator.

Operations

Displays various tasks that you can perform to manage the content, such as:

  • Send the content for review.
  • Send the content back into workflow.
  • Edit the content.
  • Preview the content.

 

 

Tip: Sorting the content list helps you quickly analyze, visualize, and understand various site content. By default, the content list displays the most recently updated content at the top of the list.  You can sort the content list according to various criteria, such as the content title, type, updated date, content creator, or moderator. To sort the content list, click the appropriate column name.
