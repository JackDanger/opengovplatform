Updating Homepage Tabs

You can update the name, content type, and content source for a Homepage Tab by performing the following steps:

  1. In the navigation menu, point to Configurable Items, and then click Manage Homepage Tabs.
    The Edit page appears.

  2. In the Block Title box, update the title for this Quicktab block.

  3. Update appropriate information in each column. The following table lists each column and its description.

Table: Description of columns in the Quicktabs section

Column

Description

Tab Title

Update the name of the Homepage Tab. This name is displayed on the home page of the OGPL Web site.

Tab Type

Select the type of content source that this Homepage Tab must use to fetch the information from the database.

Tab Content

Select the name of the content source that the Homepage Tab must use to fetch and display the content on the home page of the OGPL Web site.

If you selected View as the Tab Type, click the name of the view from the Select a view list. For example, if you want to display the most popular catalogs under the Popular Catalog Homepage Tab, in the Select a view list, click Most_Popular_Catalog.

If you selected Block as the Tab Type, click the name of the block from the Select a Block list. For example, if you want to display major events happening around the world through RSS feeds under the Major Events Homepage Tab, in the Select a Block list, click RSS Feeds Aggregator.

  1. Click Save.
    The Homepage Tabs are updated.
