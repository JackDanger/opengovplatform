Creating Rotating Panels

The Rotating Panel on the home page of the OGPL Web site is a series of images of various government-related Web sites or government-initiated programs and schemes. When a visitor clicks an image on the rotating panel, the corresponding Web site appears. A Content Creator can upload an image for a Web site to the Rotating Panel, and specify the URL for that Web site.

The following image illustrates how a Rotating Panel appears on the OGPL Web site.

To create a Rotating Panel:

  1. In the navigation menu, point to Configurable Items, and then click Create Content.
    The Create Content page appears.

  2. Click the Rotating Panel link.
    The Create Rotating Panel page appears.

  3. Enter the appropriate information in each field. The following table lists each field and its description.

Table: Description of fields in the Create Rotating Panel page

Field

Description

Title (Required)

Enter the title for this image.

Instructions

Enter the guidelines that the assigned Content Creator must follow while updating the content for this Rotating Panel. The Moderator and the Publisher reviews the content based on these guidelines.

Description

Update the description about the image and the Web site it must link to.

Image

Upload the image of the Web site for the full view of the rotating panel:

To upload an image:

  1. Click Browse.
    The File Upload dialog box appears.

  2. Click the image that you want to upload to the dataset.

  3. Click Open.
    The Image box displays the path of the file that you selected.

  4. Click Upload.
    The image is uploaded and a thumbnail of the image appears on the page.

  5. Optional: If you want to remove this image, click Remove.
    The uploaded image is removed from the database.

Smaller_image

Upload the image for the Web site that must be used for the half view of the rotating panel.

URL

Enter the URL of the Web site that the image must link to.

Menu settings

For more information about Menu settings, click here.

Body

Enter the description for this rotating panel.

Input format

Click the format in which the Content Creator can update content in the Body field. For example: Filtered HTML, Full HTML, PHP Code, or Plain Text.

Revision information – Log message

Enter the reason for creating this Rotating Panel. This message appears while viewing the revision history for this content.

URL path settings

By default, the Automatic alias check box is selected. The application automatically assigns the URL for this Rotating Panel.

If you want to assign a specific URL for this Rotating Panel, perform the following steps:

  1. Click to clear the Automatic alias check box.

  2. In the URL path settings box, type the URL that you want to assign for this Rotating Panel.

Select A Content Creator

(Required)

Click the username of the Content Creator who must update the content for this Rotating Panel.

Select A Moderator

(Required)

Click the username of the Moderator who must review the content for this Rotating Panel.

Enter the Expiry Date

Enter the date when the information in this Rotating Panel is outdated.

The content remains on the OGPL Web site until it is unpublished or deleted by the Super Administrator.

Notify Before (Days)

Enter the number of days before the expiry date when the Content Creator must receive a notification about the expiry of this Rotating Panel.

    1. Click Save.
      An entry for this Rotating Panel appears in the content list of the assigned Content Creator and Moderator.

Note: You do not have to enter values for all the fields on this page. You can enter values for the mandatory fields only and assign the Rotating Panel to a Content Creator. You or a Super Administrator can enter the appropriate values in the remaining fields after the Rotating Panel is created, reviewed, and published.

Menu link title

Enter the title of the menu item for this Rotating Panel. This menu item appears within the main menu on the OGPL Web site.

Tip: The title of this menu item can be the same as the Rotating Panel title.

Note: If you do not want to add this new Rotating Panel to a menu, leave this field blank.

Parent item

Click the menu under which the menu link for this Rotating Panel must appear. For example, if you want the menu link for this Rotating Panel to appear in the navigation bar of the OGPL Web site, click Global Navigation.

Weight

Menu links that have lesser weight appear closer to the parent item.

Assign the appropriate weight for the menu link of this Rotating Panel depending on where you want the link to appear. For example, if you selected Global Navigation as the Parent item, and you want the menu link of this Rotating Panel to appear on the navigation bar, enter 0.
