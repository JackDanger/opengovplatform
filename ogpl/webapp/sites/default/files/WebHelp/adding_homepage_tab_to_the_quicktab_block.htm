Adding Homepage Tab to the Quicktab Block

You can add a new Homepage Tab to an existing Quicktab block. You can also specify the name, content type, and content source for the quicktab.

 

To add a new Homepage Tab to a Quicktab block:

  1. In the navigation menu, point to Configurable Items, and then click Manage Homepage Tabs.
    The Edit page appears.

  2. In the Operations column, click the Add tab icon .

  3. Enter the appropriate information in each column. The following table lists each column and its description.

Table: Description of columns in the Edit page

Column

Description

Tab Title

Enter an appropriate name for the Hompage Tab. This name is displayed on the home page of the OGPL Web site. For example, if you want to create a Homepage Tab to display new Apps, documents and datasets, enter Recent Catalogs.

Tab Type

Select the appropriate type of content source that the Homepage Tab must use to fetch the information from the database.

Tab Content

Select the name of the content source that the Homepage Tab must use to fetch and display the content on the home page of the OGPL Web site.

If you selected View as the Tab Type, click the name of the view from the Select a view list. For example, if you want to display the most popular catalogs under the Popular Catalog Homepage Tab, in the Select a view list, click Most_Popular_Catalog.

If you selected Block as the Tab Type, click the name of the block from the Select a Block list. For example, if you want to display major events happening around the world through RSS feeds under the Major Events Homepage Tab, in the Select a Block list, click RSS Feeds Aggregator.

  1. Click Save.
    The Homepage Tab is added to the Quicktab block and the Homepage Tab appears on the home page of the OGPL Web site.
