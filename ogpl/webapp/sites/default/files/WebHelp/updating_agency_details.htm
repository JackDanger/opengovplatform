Updating Agency Details

You can update agency details, such as the agency name or short name, and agency settings, such as the menu settings and URL settings for an agency. After updating the appropriate settings, you can preview the updated agency details before saving the changes that you made.

To update the agency details:

  1. In the navigation menu, point to the Management link and click Agency Management.
    The Agency Management page appears.

  2. In the row of the agency that you want to edit, click the edit link.
    The agency details appear.

  3. Update the appropriate agency settings. The following table lists each field and its description.
    Table: Description of fields in the Edit tab

Field

Description

Agency Name  

Update the name for this agency. For example, Forest and Mining.

Agency Short Name

Update the acronym or abbreviation for this agency. For example, if the agency name is Forest and Mining, you can enter FM in this box.

Menu settings

Click this link to specify menu settings, such as the menu link title, parent item, and weight.

Menu link title

Enter the title of the menu item for this agency. This menu item appears within the main menu on the OGPL Web site.

Tip: The title of this menu item can be the same as the page title.

Note: If you do not want to add this new agency to a menu, leave this box blank.

Parent item

Click the menu under which the menu link for this agency must appear.

For example, if you want the menu link for this agency to appear in the navigation bar of the OGPL Web site, click Global Navigation.

Weight

Assign the appropriate position for the menu link of this agency depending on where you want the link to appear. The higher the weight value, the lower will be the position of the agency in the menu list.

For example, if you selected Global Navigation as the Parent item, and you want the menu link of this block to appear on the navigation bar, enter 0.

URL path settings

By default, the Automatic alias check box is selected. The application automatically assigns the URL for this agency.

If you want to assign a specific URL for this agency, perform the following steps:

  1. Click to clear the Automatic alias check box.

  2. In the URL path settings box, type the URL that you want to assign for this agency.

 

  1. Optional: To see the updated agency details before saving the changes, click Preview.

  1. Click Save.
    The agency details are updated in the OGPL Control Panel application.
