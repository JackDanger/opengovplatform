Configuring the Number of Items to Display in the Recent Catalogs Homepage tab

The Recent Catalogs Homepage tab appears on the home page of the OGPL Web site. This Homepage tab displays a list of datasets that were recently updated or added to the OGPL Web site. By default, the Recent Catalogs Homepage tab displays three types of datasets - Apps, Datasets, and Documents. For more information on how to configure the number of items to display the three types of datasets, see Managing Homepage Tabs.

You can configure the number of items that must appear in the Recent Catalogs Hoempage tab by performing the following steps:

  1. Point to Site building, and then click Views.
    The Views page appears.

  2. Scroll down until you see Most_Recent_Catalog, and then click the Edit link.

  3. In the Basic settings section, click the number displayed next to Items to display.

  4. Below the Basic setting section, in the Defaults: Items to display box, update the number of items that you want to display in the Recent Catalogs Homepage tab.

  5. Click Update.

  6. Click Save.

The Recent Catalogs Homepage tab displays the number of items that you specified.
