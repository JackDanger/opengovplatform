Deleting Color Themes

You can delete a color theme from the OGPL Control Panel application and OGPL Web site. You cannot restore a deleted color theme.

Note: You cannot delete the color themes that are available by default, such as Grey, Blue, Pink, and Green.

To delete a color theme:

  1. In the navigation menu, point to Configurable Items, Manage Header Content, and then click Manage Theme Files.
    The Theme Files page appears and displays the list of the color themes that you created.

  2. In the row of the theme that you want to delete, click the delete link.
    A confirmation message appears.

  3. Click Delete.
    The color theme is deleted and its color theme icon disappears from the OGPL Web site.
