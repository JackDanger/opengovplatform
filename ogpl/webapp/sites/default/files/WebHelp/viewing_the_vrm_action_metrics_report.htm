Viewing the VRM Action Metrics Report

The VRM Action Metrics report displays the percentage of feedback that is currently in a specific action status. This report enables you to analyze the feasibility of the feedback that the visitors provided. By default, the VRM Action Metrics report displays the metrics for all the feedback received till date.

To view the VRM Action Metrics report, in the navigation menu, point to Reports, VRM Agency Reports, and then click VRM Action Metrics.

Tip: To view the number of feedback for each action status, place the mouse pointer over the appropriate action status on the pie chart.

You can also view the VRM Action Metrics report for a specific month and feedback category by performing the following steps:

  1. In the Select Year list, click the year for which you want to analyze the feedback action metrics.

  2. In the Select Month list, click the month for which you want to analyze the feedback action metrics.

  3. In the Select Category list, click the feedback category for which you want to analyze the feedback action metrics.

  4. Click Submit.
    The VRM Action Metrics report displays the information for the month and feedback category that you selected.
