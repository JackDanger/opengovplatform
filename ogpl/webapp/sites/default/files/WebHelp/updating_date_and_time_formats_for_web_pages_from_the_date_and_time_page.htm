Updating Date and Time Formats for Web Pages

You can change the formats for date and time that appear on various pages of the OGPL Web site such as Feedback Details and Agency Wise page. When you change the format of date and time for any of the date and time formats types, the date and time format of the page that uses this format type gets updated automatically. For example, the Agency Wise page displays the date and time of dataset submission in the format that you specified for the front date format type in the Date and Time page. If you change the date and time format for the front date format type from dd-mm-yyyy format to d-month-yyyy – h:mm format, the date and time on the Agency Wise page is updated and displays the date and time in the new format that you specified. To update Date and Time format for various Web pages:

  1. In the Drupal menu, point to Site configuration, Date and time, formats, and then click Configure.
    The Date And Time page appears.

  2. In the Date formats section, click the appropriate information. The following table lists each field and its description:

Table: List of format types in the Date formats section

Format Type

Description

Long date format

Click the appropriate date and time format that you want to display on the pages that use this date and time format type to display date and time on the OGPL Web site. For example, Tuesday, March 6, 2012 – 18:53.

Medium date format

Click the appropriate date and time format that you want to display on the pages that use this date and time format type to display date and time on the OGPL Web site. For example, 06-03-2012 18:53:49.

Short date format

Click the appropriate date and time format that you want to display on the pages that use this date and time format type to display date and time on the OGPL Web site. For example, 06-03-2012 18:53.

front date format

Click the appropriate date and time format that you want to display on the pages that use this date and time format type to display date and time on the OGPL Web site. For example, 06-03-2012.

notify date format

Click the appropriate date and time format that you want to display on the pages that use this date and time format type to display date and time on the OGPL Web site. For example, 2012-03-06.

 

  1. Click Save configuration.
    The date and time format for the pages that use this format type is updated.
