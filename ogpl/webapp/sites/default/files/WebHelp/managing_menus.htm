Managing Menus and Menu Items

A menu is a collection of links and menu items that you can use to navigate between various pages of the OGPL Web site. The Manage Menus page allows you to control and customize the menus that appear on the OGPL Web site. The List Menus page displays a list of all menus that appear on the OGPL Web site.

Table: Description of menus in the List Menus page

Menu

Description

Bottom Links

Contains menu items that appear as bottom links on the OGPL Web site.

Global Navigation

Contains menu items that appear on the home page and link to various pages within the OGPL Web site.

Home Page Header Menu

Contains menu items that appear on the header section of the home page.

Inner Page Footer Menu

Contains menu items that appear in the footer section of the inner pages.

Metrics Navigation

Contains menu items for second-level navigation for metrics.

You can perform the following tasks to manage the menus that appear on the OGPL Web site:

  • Update menus. For more information see, Updating Menus.

  • Configure the hierarchy of menu items. For more information see, Arranging Menu Items.

  • Add menu items to a menu. For more information see, Adding a Menu Item to a Menu.

  • Update menu items for a menu. For more information see, Updating Menu Item.

  • Delete menu items from a menu. For more information see, Deleting a Menu Item.
