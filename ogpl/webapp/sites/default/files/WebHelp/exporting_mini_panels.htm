Exporting Mini Panels

You can export an existing Mini Panel to create  its export definition. This export definition is required if you want to import this Mini Panel in the future.

To export a Mini Panel:

  1. In the Drupal menu, point to Site building, Mini panels, and then click List.
    The Mini Panels page appears.
    The List tab displays all the Mini Panels that are configured in the OGPL Web site.

  2. In the row of the Mini panel that you want to export, click Export.
    The Export Mini Panel page displays the export definition for this Mini Panel.

  3. Optional: If you want to import this Mini Panel, copy the export definition. For more information, see Importing Mini Panels.
